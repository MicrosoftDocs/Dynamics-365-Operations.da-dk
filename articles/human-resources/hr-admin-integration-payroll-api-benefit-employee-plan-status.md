# <a name="benefit-employee-plan-status"></a>Frynsegodestatus for medarbejderplan

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling af frynsegodestatus for medarbejderplan, der er angivet for Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Angivet indstilling: Procestype

Fysisk navn: mshr_hrmCompProcessType

Denne fasttekst indeholder indstillingen, der er angivet for procestype i virtuelle enheder for Human Resources.

| Værdi | Navn | Betegnelse |
| --- | --- | --- |
| 200000000 | None | Ingen. |
| 200000001 | Udvalgt | Udvalgt. |
| 200000002 | Frafaldet | Frafaldet. |
