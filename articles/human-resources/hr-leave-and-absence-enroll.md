# <a name="assign-workers-to-a-leave-plan"></a><span data-ttu-id="1271c-103">Tildele arbejdere til en orlovsplan</span><span class="sxs-lookup"><span data-stu-id="1271c-103">Assign workers to a leave plan</span></span>

## <a name="assign-a-plan-to-a-group-of-employees"></a><span data-ttu-id="1271c-105">Tildele en plan til en gruppe af medarbejdere</span><span class="sxs-lookup"><span data-stu-id="1271c-105">Assign a plan to a group of employees</span></span>

1. <span data-ttu-id="1271c-106">På siden **Orlov og fravær** skal du vælge den plan, du vil tildele.</span><span class="sxs-lookup"><span data-stu-id="1271c-106">On the **Leave and absence** page, select the plan you want to assign.</span></span>

3. <span data-ttu-id="1271c-108">Juster følgende indstillinger efter behov:</span><span class="sxs-lookup"><span data-stu-id="1271c-108">Adjust the following settings as appropriate:</span></span>

   - <span data-ttu-id="1271c-109">**Startdato for tilmeldingsansættelse**</span><span class="sxs-lookup"><span data-stu-id="1271c-109">**Enroll on employment start date**</span></span>

   - <span data-ttu-id="1271c-111">**Slutdato**</span><span class="sxs-lookup"><span data-stu-id="1271c-111">**End date**</span></span>
   - <span data-ttu-id="1271c-112">**Niveaubasis**</span><span class="sxs-lookup"><span data-stu-id="1271c-112">**Tier basis**</span></span>
