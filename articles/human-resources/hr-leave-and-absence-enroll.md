# <a name="assign-workers-to-a-leave-plan"></a><span data-ttu-id="f84ef-103">Tildele arbejdere til en orlovsplan</span><span class="sxs-lookup"><span data-stu-id="f84ef-103">Assign workers to a leave plan</span></span>

   - <span data-ttu-id="f84ef-111">**Slutdato**</span><span class="sxs-lookup"><span data-stu-id="f84ef-111">**End date**</span></span>
   - <span data-ttu-id="f84ef-112">**Niveaubasis**</span><span class="sxs-lookup"><span data-stu-id="f84ef-112">**Tier basis**</span></span>
