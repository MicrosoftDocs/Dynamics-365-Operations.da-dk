# <a name="payroll---gender-option-set"></a>Løn – Den indstilling, der er angivet for Køn

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling for Køn, der er angivet for Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Angivet indstilling: indstillingen Køn

Denne fasttekst indeholder den indstilling, der er angivet for medarbejderens køn. Dette er tilgængeligt for opsætning af indstillingen mshr_hcmpersongender.

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 | None | Der er ikke angivet noget køn. |
| 200000001 | Mand | Mand. |
| 200000002 | Kvinde | Kvinde. |
| 200000003 | Ikke-specifik | Valg for et ikke-bestemt køn. |
