# <a name="generate-payroll-entities"></a>Generere lønenheder

Brug denne OData-funktion til at generere de enheder, der er nødvendige for lønintegration. Hvis der foretages ændringer af disse enheder i Human Resources, f.eks. tilføjelse af brugerdefinerede felter, kan denne funktion kaldes igen for at opdatere metadata for hver enhed. Svaret indeholder et handlings-id, som du kan overvåge, så du ved, hvornår oprettelsesprocessen er fuldført.

**Anmodning**

```http
GET [Organizaton URI]/api/data/v9.1/RefreshHumanResourcesVirtualEntities
```

**brødtekst**

```json
{
    "PhysicalNames" : ["PayrollEmployeeEntity", "HcmWorkerBaseEntity", "PayrollPositionEntity", "PayrollPositionJobEntity", "PayrollWorkerAddressEntity", "HcmJobDetailEntity", "HcmCompFixedPlanTableEntity", "PayrollFixedCompensationPlanEntity", "HcmEmploymentDetailEntity"]
}
```

**Svar**

```json
{
    "AsyncOperationId": "8b98d338-f939-4c86-9a91-80b76b6ab2ea"
}
```

**Anmodning**

```http
GET [Organizaton URI]/api/data/v9.1/mshr_hrvirtualentitycatalogs?$filter=mshr_hasbeengenerated eq true
```
