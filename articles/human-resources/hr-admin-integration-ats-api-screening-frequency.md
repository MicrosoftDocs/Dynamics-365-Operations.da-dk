# <a name="screening-frequency"></a>Screeningfrekvens


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver screeningfrekvens-indstilling, der er angivet til Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmfrequencyunit

Denne fasttekst indeholder et sæt indstillinger med værdier til screeningfrekvens. 

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 kun en gang | Screeningen er kun påkrævet én gang. |
| 200000001 dagligt | Screeningfrekvensen beregnes i dage. |
| 200000002 ugentlig | Screeningfrekvensen beregnes i uger. |
| 200000003 månedlig | Screeningfrekvensen beregnes i måneder. |
| 200000004 årlig | Screeningfrekvensen beregnes i år. |
