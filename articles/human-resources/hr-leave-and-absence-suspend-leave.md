# <a name="suspend-leave"></a><span data-ttu-id="e71ad-103">Stop orlov midlertidigt</span><span class="sxs-lookup"><span data-stu-id="e71ad-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="e71ad-104">Du kan suspendere en medarbejders orlov for at stoppe med at lade periodiseringer blive behandlet for de valgte orlovstyper.</span><span class="sxs-lookup"><span data-stu-id="e71ad-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="e71ad-105">Suspendere orlov og fravær for en medarbejder</span><span class="sxs-lookup"><span data-stu-id="e71ad-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="e71ad-106">Vælg **Orlov** i medarbejderens post.</span><span class="sxs-lookup"><span data-stu-id="e71ad-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="e71ad-107">Vælg **Suspender orlov**.</span><span class="sxs-lookup"><span data-stu-id="e71ad-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="e71ad-108">Vælg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="e71ad-108">Select **New**.</span></span>

4. <span data-ttu-id="e71ad-109">Vælg **Orlovstype** sammen med **Startdato** og **Slutdato** for suspensionen i dialogboksen **Suspender periodisering af orlov**.</span><span class="sxs-lookup"><span data-stu-id="e71ad-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="e71ad-110">Du kan også føje en **Kommentar** til suspensionen.</span><span class="sxs-lookup"><span data-stu-id="e71ad-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="e71ad-111">Hvis periodiseringer behandles, mens medarbejderens orlov er suspenderet, foretages der ingen periodisering af de suspenderede orlovstyper.</span><span class="sxs-lookup"><span data-stu-id="e71ad-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
