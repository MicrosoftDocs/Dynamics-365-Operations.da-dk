# <a name="completion-status"></a>Status for færdiggørelse

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling for status for færdiggørelse, der er angivet for Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmcompletionstatus

Denne fasttekst indeholder et den indstilling, der er angivet for statusværdier for kandidatscreeninger. 

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 | Ikke færdig | Kandidaten har endnu ikke fuldført screeningen. |
| 200000001 | Bestået | Kandidaten har bestået screeningen. |
| 200000002 | Mislykket | Kandidaten har ikke bestået screeningen. |
