# <a name="payroll---employment-type-option-set"></a>Løn – indstilling for ansættelsestype

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling for ansættelsestypen, der er angivet for Dynamics 365 Human Resources.

## <a name="option-set-employment-type"></a>Indstilling: Ansættelsestype

Denne fasttekst indeholder den indstilling, der er angivet for medarbejderens ansættelsestype. Dette er tilgængeligt for opsætning af indstillingen mshr_hcmemploymenttype.

| Værdi | Navn | Betegnelse |
| --- | --- | --- |
| 200000000 | Medarbejder | Medarbejder |
| 200000001 | Kontrahent | Kontrahent |
