# <a name="period-unit"></a>Periodeenhed

Dette emne beskriver den indstilling for periodeenheden, der er angivet for Dynamics 365 Human Resources.

Fysisk navn: mshr_periodunit

Denne fasttekst indeholder den indstillinger, der er angivet for værdier for måleenheder for tidsperioder.

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 | Dage | Dag. |
| 200000001 | Måneder | Måned. |
| 200000002 | År | År. |
