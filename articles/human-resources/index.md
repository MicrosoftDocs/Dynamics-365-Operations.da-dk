# <a name="human-resources-home-page"></a>Startside for Human Resources

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Dynamics 365 Human Resources giver den indsigt i arbejdsstyrken, du skal bruge til at opbygge databaserede medarbejdererfaringer på tværs af flere områder, herunder:

- Kompensation
- Frynsegoder
- Orlov og fravær
- Overholdelse
- Lønintegration
- Feedback på ydeevne
- Uddannelse og certificering
- Selvbetjeningsprogrammer

Human Resources forbinder personer og driftsdata for at hjælpe dig med at optimere udgifterne til personale og tage hånd om dine medarbejdere.
