# <a name="set-up-employment-types"></a>Konfigurere ansættelsestyper

[!include [banner](includes/preview-feature.md)]

Opret ansættelsestyper i Dynamics 365 Human Resources, så de svarer til medarbejderne.

1. I arbejdsområdet **Frynsegodeadministration** skal du vælge **Ansættelsestyper** under **Konfiguration**.

2. Vælg **Ny**.

3. Angiv værdier for følgende felter:

   | Felt | Beskrivelse |
   | --- | --- |
   | Medarbejdertype | Entydig identifikation for ansættelsestypen. |
   | Beskrivelse | En beskrivelse af ansættelsestypen. |

4. Vælg **Gem**.
