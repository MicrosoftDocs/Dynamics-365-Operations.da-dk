# <a name="no-yes"></a>Nej Ja

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling for Nej Ja, der er angivet for Dynamics 365 Human Resources.

## <a name="option-set-no-yes"></a>Den indstilling, der er angivet: Nej Ja

Fysisk navn: mshr_noyes

Denne fasttekst indeholder indstillinger, der er angivet for typiske booleske egenskaber i virtuelle enheder for Human Resources.

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 | Ingen | Nej |
| 200000001 | Ja | Ja. |
