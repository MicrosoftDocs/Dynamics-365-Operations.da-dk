# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="16eeb-103">Skifte arbejdsområdenavn for medarbejderselvbetjening</span><span class="sxs-lookup"><span data-stu-id="16eeb-103">Change Employee self service workspace name</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="16eeb-104">Hvis du har frivillige eller andre, der ikke er medarbejdere, kan det være en god ide at ændre navnet på arbejdsområdet **Medarbejderselvbetjening**.</span><span class="sxs-lookup"><span data-stu-id="16eeb-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="16eeb-105">Du kan ændre dette arbejdsområde til **Selvbetjening** i stedet.</span><span class="sxs-lookup"><span data-stu-id="16eeb-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="16eeb-106">Hvis du ændrer navnet på arbejdsområdet **Medarbejderselvbetjening**, ændres også det menupunkt, der bruges internt af Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="16eeb-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="16eeb-107">Hvis du tidligere har anvendt sikkerhedstilpasninger på menupunktet **HcmMedarbejderSelvbetjeningArbejdsområde**, anbefales det, at du anvender de samme ændringer i **HcmSelvbetjeningArbejdsområde** for at bevare pariteten.</span><span class="sxs-lookup"><span data-stu-id="16eeb-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

2. <span data-ttu-id="16eeb-109">Vælg fanen **Medarbejderselvbetjening**.</span><span class="sxs-lookup"><span data-stu-id="16eeb-109">Select the **Employee self-service** tab.</span></span>
