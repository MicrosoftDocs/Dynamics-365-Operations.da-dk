# <a name="custom-fields"></a>Brugerdefinerede felter

Du kan føje brugerdefinerede felter til sider i Dynamics 365 Human Resources. Du kan finde flere oplysninger om, hvordan du opretter brugerdefinerede felter på en side, i [Oprette og arbejde med brugerdefinerede felter](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Følgende enheder i Common Data Service understøtter brugertilpassede felter, der oprettes i Human Resources:

- Arbejdstråd
- Etnisk oprindelse
- Veteranstatus
- Sprogkode
- Stilling
- Jobtype
- Jobfunktion
- Stilling
- Stillingstype

Sørg for at aktivere eventuelle brugerdefinerede felter, du opretter, så du kan bruge dem i Common Data Service. Du kan finde flere oplysninger [Visning af brugerdefinerede felter på dataenheder](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
