| 200000000 | None | Ingen. |
| 200000001 | Årlig | Årligt. |
| 200000002 | Jubilæum | Jubilæum. |
| 200000003 | Ikke-cyklisk | Ikke-cyklisk. |
| 200000004 | ½ år | Halvårligt. |
