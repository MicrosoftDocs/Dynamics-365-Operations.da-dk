# <a name="contact-type"></a>Kontakttype

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling for kontakttypen, der er angivet for Dynamics 365 Human Resources.

Fysisk navn: mshr_logisticselectronicaddressmethodtype

Denne fasttekst indeholder den indstilling, der er angivet for værdier for kontaktpersontyper, der ansøger. 

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 | None | Der er ikke valgt en type. |
| 200000001 | Telefon | Telefonnummer. |
| 200000002 | E-mailadresse | E-mailadresse. |
| 200000003 | URL | URL-adresse for websted. |
| 200000004 | Telex | Telexnummer. |
| 200000005 | Telefax | Faxnummer. |
| 200000006 | Facebook | Facebook-konto. Identificeret af bruger-id. |
| 200000007 | Twitter | Twitter-konto. Identificeret af @brugernavn. |
| 200000008 | LinkedIn | LinkedIn-konto. Identificeret af brugernavn. |
