# <a name="education-credit-basis"></a>Uddannelseskreditgrundlag

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling for kreditgrundlag for uddannelse, der er angivet for Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmeducationcreditbasis

Denne fasttekst indeholder det værdisæt af værdier for uddannelseskreditgrundlaget, der er angivet for kandidatens uddannelsespost.

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 | Tom | Der er ikke valgt en værdi. |
| 200000001 | Semester | Semester. |
| 200000002 | Kvartal | Kvartal. |
| 200000003 | Trimester | Trimester. |
| 200000004 | Periode | Periode. |
| 200000005 | Anden | Andet. |
