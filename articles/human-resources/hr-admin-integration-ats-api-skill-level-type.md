# <a name="skill-level-type"></a>Færdighedsniveautype

Dette emne beskriver den indstilling for færdighedsniveautypen, der er angivet for Dynamics 365 Human Resources.

Fysisk navn: mshr_hrmskillleveltype

Denne fasttekst indeholder det værdisæt af indstillinger, der kategoriserer det niveau, der er tildelt en kandidats færdighed.

| Værdi | Mærkat | Betegnelse |
| --- | --- | --- |
| 200000000 | Realiseret | Faktisk færdighedsniveau. |
| 200000001 | Målsætning | Den værdi, der er mål for færdighedsniveauet. |
