# <a name="regulatory-job-category"></a>Kategori af lovpligtigt job

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling til kategori af lovpligtigt job, der er angivet for Dynamics 365 Human Resources.

Fysisk navn: mshr_hcmregulatoryjobcategory

Denne fasttekst angiver den indstilling, der er angivet for angivelse af værdier for lovpligtig jobkategori. Dette findes i den eksisterende indstilling for mshr_hcmregulatoryjobcategory.

| Værdi | Mærkat | Beskrivelse |
| --- | --- | --- |
| 200000000 | None | Ingen. |
| 200000001 | Ledelse | Chefer/overordnede medarbejdere og ledere. |
| 200000002 | Administrator | Første/mellemste niveau og ledere. |
| 200000003 | Professionel | Professionelle. |
| 200000004 | Tekniker | Teknikere. |
| 200000005 | Salgsmedarbejder | Salgsmedarbejdere. |
| 200000006 | Administrativ | Administrative medarbejdere. |
| 200000007 | Akkord-arbejder | Akkord-arbejdere. |
| 200000008 | Operativ medarbejder | Operative medarbejdere. |
| 200000009 | Arbejdsmand | Arbejdsmænd/hjælpere. |
| 200000010 | Servicemedarbejder | Servicemedarbejdere. |
