# <a name="award-type"></a>Bonustype

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dette emne beskriver den indstilling for bonustypen, der er angivet for Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Angivet indstilling: Bonustype

Fysisk navn: mshr_HrmCompVarAwardEmplType

Denne fasttekst indeholder den indstilling, der er angivet for bonustypen for den variable kompensation i virtuelle enheder for Human Resources.

| Værdi | Navn | Betegnelse |
| --- | --- | --- |
| 200000000 | Generel bonus | Generel bonus. |
| 200000001 | Engangsbeløb som fast løn | Engangsbeløb som fast løn. |
