# <a name="install-the-retail-pos-layout-designer"></a><span data-ttu-id="bfc9f-103">Installere layoutdesigneren til Retail POS</span><span class="sxs-lookup"><span data-stu-id="bfc9f-103">Install the Retail POS Layout designer</span></span>

[!include[banner](includes/banner.md)]


<span data-ttu-id="bfc9f-104">Du kan bruge ét-klik-designer til at designe forskellige Retail Modern POS (MPOS) og Cloud POS-layout, i enten liggende eller stående tilstand, til butikker, kasseapparater, kasserer og ledere.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-104">You can use the one-click designer to design different Retail Modern POS (MPOS) and Cloud POS layouts, in either Landscape mode or Portrait mode, for stores, registers, cashiers, and managers.</span></span>

<span data-ttu-id="bfc9f-105">Det grafiske design af brugergrænsefladen for MPOS og Cloud POS styres af pengeskuffelayoutet.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-105">The graphical design interface for MPOS or Cloud POS is controlled by the till layout.</span></span> <span data-ttu-id="bfc9f-106">Et layout styrer placeringen af forskellige objekter.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-106">A layout controls the position of various objects.</span></span> <span data-ttu-id="bfc9f-107">Af eksempler kan nævnes det samlede layout, layout for varegitter, kundelayout og betalingslayout samt layoutet for forskellige menuknapper.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-107">Examples include the total layout, the item grid layout, the customer layout, the payment layout, and the layout of various menu buttons.</span></span> <span data-ttu-id="bfc9f-108">Layout omfatter også det overordnede udseende af den salgsgrænseflade, som arbejderne kan se.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-108">Layouts also include the overall appearance of the sales interface that is presented to workers.</span></span>

## <a name="install-the-one-click-designer"></a><span data-ttu-id="bfc9f-109">Installere enkelt klik-designer</span><span class="sxs-lookup"><span data-stu-id="bfc9f-109">Install the one-click designer</span></span>
1.  <span data-ttu-id="bfc9f-110">Brug menuen øverst til venstre i Microsoft Dynamics 365 for Retail til at navigere til **Detail** **og handel** &gt; **Konfiguration af kanal** &gt; **POS-opsætning** &gt; **POS** &gt; **Skærmlayout**.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-110">In Microsoft Dynamics 365 for Retail, use the menu in the upper left to navigate to **Retail** **and commerce** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Screen layouts**.</span></span>
2.  <span data-ttu-id="bfc9f-111">Vælg et layout, der har programtypen **Modern POS til Windows** eller **Cloud POS**, og klik derefter på **Layoutdesigner**.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
3.  <span data-ttu-id="bfc9f-112">Klik på **Åbn** på meddelelseslinjen nederst i vinduet Internet Explorer for at starte ét klik-designeren.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-112">On the notification bar that appears at the bottom of the Internet Explorer window, click **Open** to install the one-click designer.</span></span> <span data-ttu-id="bfc9f-113">(Meddelelseslinjen vises muligvis et andet sted i andre browsere).</span><span class="sxs-lookup"><span data-stu-id="bfc9f-113">(The notification bar might appear in a different place in other browsers.)</span></span>
4.  <span data-ttu-id="bfc9f-114">I **Programkørsel - Sikkerhedsadvarsel**-meddelelsen, der vises, skal du klikke på **Kør** for at installere Retail designerværten.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-114">In the **Application Run - Security Warning** message box that appears, click **Run** to install the Retail designer host.</span></span> <span data-ttu-id="bfc9f-115">Statusindikatoren viser forløbet af installationen.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-115">A progress indicator shows the progress of the installation.</span></span>
5.  <span data-ttu-id="bfc9f-116">Når installationen er fuldført, skal du angive dit brugernavn og din adgangskode til Microsoft Dynamics 365 for Retail på siden **Log på** og derefter klikke på **Log på** for at starte designeren.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-116">After the installation is completed, on the **Sign in** page, enter your Microsoft Dynamics 365 for Retail user name and password, and then click **Sign in** to start the designer.</span></span>
6.  <span data-ttu-id="bfc9f-117">Når dine legitimationsoplysninger er valideret, og designeren starter, kan du designe dit eget layout eller redigere det eksisterende layout.</span><span class="sxs-lookup"><span data-stu-id="bfc9f-117">After your credentials are validated and the designer starts, you can design your own layout or modify the existing layout.</span></span> <span data-ttu-id="bfc9f-118">[![Layout i ét klik-designeren](./media/screenlayoutdesign_mposdownload-1024x664.png)](./media/screenlayoutdesign_mposdownload.png)</span><span class="sxs-lookup"><span data-stu-id="bfc9f-118">[![Layout in the one-click designer](./media/screenlayoutdesign_mposdownload-1024x664.png)](./media/screenlayoutdesign_mposdownload.png)</span></span>
