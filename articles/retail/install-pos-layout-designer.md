# <a name="install-the-retail-point-of-sale-pos-layout-designer"></a><span data-ttu-id="35f55-103">Installere layoutdesigneren til Retail POS</span><span class="sxs-lookup"><span data-stu-id="35f55-103">Install the Retail point of sale (POS) layout designer</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="35f55-104">Du kan bruge ét-klik-designer til at designe forskellige Retail Modern POS (MPOS) og Cloud POS-layout, i enten liggende eller stående tilstand, til butikker, kasseapparater, kasserer og ledere.</span><span class="sxs-lookup"><span data-stu-id="35f55-104">You can use the one-click designer to design different Retail Modern POS (MPOS) and Cloud POS layouts, in either Landscape mode or Portrait mode, for stores, registers, cashiers, and managers.</span></span>

<span data-ttu-id="35f55-105">Det grafiske design af brugergrænsefladen for MPOS og Cloud POS styres af pengeskuffelayoutet.</span><span class="sxs-lookup"><span data-stu-id="35f55-105">The graphical design interface for MPOS or Cloud POS is controlled by the till layout.</span></span> <span data-ttu-id="35f55-106">Et layout styrer placeringen af forskellige objekter.</span><span class="sxs-lookup"><span data-stu-id="35f55-106">A layout controls the position of various objects.</span></span> <span data-ttu-id="35f55-107">Af eksempler kan nævnes det samlede layout, layout for varegitter, kundelayout og betalingslayout samt layoutet for forskellige menuknapper.</span><span class="sxs-lookup"><span data-stu-id="35f55-107">Examples include the total layout, the item grid layout, the customer layout, the payment layout, and the layout of various menu buttons.</span></span> <span data-ttu-id="35f55-108">Layout omfatter også det overordnede udseende af den salgsgrænseflade, som arbejderne kan se.</span><span class="sxs-lookup"><span data-stu-id="35f55-108">Layouts also include the overall appearance of the sales interface that is presented to workers.</span></span>

## <a name="install-the-one-click-designer"></a><span data-ttu-id="35f55-109">Installere enkelt klik-designer</span><span class="sxs-lookup"><span data-stu-id="35f55-109">Install the one-click designer</span></span>

1. <span data-ttu-id="35f55-110">I Retail skal du bruge menuen øverst til venstre til at navigere til **Detail** **og handel** &gt; **Konfiguration af kanal** &gt; **POS-opsætning** &gt; **POS** &gt; **Skærmlayout**.</span><span class="sxs-lookup"><span data-stu-id="35f55-110">In Retail, use the menu in the upper left to navigate to **Retail** **and commerce** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Screen layouts**.</span></span>
2. <span data-ttu-id="35f55-111">Vælg et layout, der har programtypen **Modern POS til Windows** eller **Cloud POS**, og klik derefter på **Layoutdesigner**.</span><span class="sxs-lookup"><span data-stu-id="35f55-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
3. <span data-ttu-id="35f55-112">Klik på **Åbn** på meddelelseslinjen nederst i vinduet Internet Explorer for at starte ét klik-designeren.</span><span class="sxs-lookup"><span data-stu-id="35f55-112">On the notification bar that appears at the bottom of the Internet Explorer window, click **Open** to install the one-click designer.</span></span> <span data-ttu-id="35f55-113">(Meddelelseslinjen vises muligvis et andet sted i andre browsere).</span><span class="sxs-lookup"><span data-stu-id="35f55-113">(The notification bar might appear in a different place in other browsers.)</span></span>
4. <span data-ttu-id="35f55-114">I **Programkørsel - Sikkerhedsadvarsel**-meddelelsen, der vises, skal du klikke på **Kør** for at installere Retail designerværten.</span><span class="sxs-lookup"><span data-stu-id="35f55-114">In the **Application Run - Security Warning** message box that appears, click **Run** to install the Retail designer host.</span></span> <span data-ttu-id="35f55-115">Statusindikatoren viser forløbet af installationen.</span><span class="sxs-lookup"><span data-stu-id="35f55-115">A progress indicator shows the progress of the installation.</span></span>
5. <span data-ttu-id="35f55-116">Når installationen er fuldført, skal du på siden **Log på** angive dit Retail-brugernavn og din adgangskode og derefter klikke på **Log på** for at starte designeren.</span><span class="sxs-lookup"><span data-stu-id="35f55-116">After the installation is completed, on the **Sign in** page, enter your Retail user name and password, and then click **Sign in** to start the designer.</span></span>
6. <span data-ttu-id="35f55-117">Når dine legitimationsoplysninger er valideret, og designeren starter, kan du designe dit eget layout eller redigere det eksisterende layout.</span><span class="sxs-lookup"><span data-stu-id="35f55-117">After your credentials are validated and the designer starts, you can design your own layout or modify the existing layout.</span></span>

- <span data-ttu-id="35f55-120">Når du klikker på **Designer**, bliver du ikke bedt om at downloade (eller køre) installationsprogrammet, eller dine aktuelle sikkerhedsindstillinger tillader ikke download af filen.</span><span class="sxs-lookup"><span data-stu-id="35f55-120">When you click **Designer**, the prompt to download (or run) the installer doesn't appear, or your current security settings don't allow you to download the file.</span></span>

    - <span data-ttu-id="35f55-124">I Internet Explorer skal du føje Retail URL-adressen til listen over websteder, der er tillid til.</span><span class="sxs-lookup"><span data-stu-id="35f55-124">In Internet Explorer, add the Retail URL to your trusted sites.</span></span> <span data-ttu-id="35f55-125">Klik på **Indstillinger** &gt; **Indstillinger** &gt; **Sikkerhed** &gt; **Websteder, du har tillid til** &gt; **Websteder** &gt; **Tilføj**.</span><span class="sxs-lookup"><span data-stu-id="35f55-125">Click **Settings** &gt; **Options** &gt; **Security** &gt; **Trusted sites** &gt; **Sites** &gt; **Add**.</span></span>

- <span data-ttu-id="35f55-126">Programmet starter ikke, og du bliver bedt om at kontakte leverandøren.</span><span class="sxs-lookup"><span data-stu-id="35f55-126">The program doesn't start, and you're instructed to contact the vendor.</span></span>

    <span data-ttu-id="35f55-127">**Løsning:** I Internet Explorer skal du føje Retail URL-adressen til listen over websteder, der er tillid til.</span><span class="sxs-lookup"><span data-stu-id="35f55-127">**Solution:** In Internet Explorer, add the Retail URL to your trusted sites.</span></span> <span data-ttu-id="35f55-128">Klik på **Indstillinger** &gt; **Indstillinger** &gt; **Sikkerhed** &gt; **Websteder, du har tillid til** &gt; **Websteder** &gt; **Tilføj**.</span><span class="sxs-lookup"><span data-stu-id="35f55-128">Click **Setting** &gt; **Options** &gt; **Security** &gt; **Trusted sites** &gt; **Sites** &gt; **Add**.</span></span>
