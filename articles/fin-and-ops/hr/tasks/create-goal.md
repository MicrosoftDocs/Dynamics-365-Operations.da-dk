[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="24cb5-104">Du kan oprette mål, føje aktiviteter til mål, spore målinger og knytte mål til en evaluering.</span><span class="sxs-lookup"><span data-stu-id="24cb5-104">You can create goals, add activities to goals, track measurements, and link the goals to a review.</span></span> <span data-ttu-id="24cb5-105">Denne procedure blev oprettet ved hjælp af demodatafirmaet USMF.</span><span class="sxs-lookup"><span data-stu-id="24cb5-105">This procedure was created using the USMF demo data company.</span></span> <span data-ttu-id="24cb5-106">Denne procedure er til en funktion, der blev tilføjet i Dynamics 365 for Operations, version 1611.</span><span class="sxs-lookup"><span data-stu-id="24cb5-106">This procedure is for a feature that was added in Dynamics 365 for Operations, version 1611.</span></span>

1. <span data-ttu-id="24cb5-107">Gå til Alle arbejdsområder > Medarbejderselvbetjening.</span><span class="sxs-lookup"><span data-stu-id="24cb5-107">Go to All workspaces > Employee self service.</span></span>
2. <span data-ttu-id="24cb5-108">Klik på Vis alle mål</span><span class="sxs-lookup"><span data-stu-id="24cb5-108">Click View all goals</span></span>
3. <span data-ttu-id="24cb5-109">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="24cb5-109">Click New.</span></span>
4. <span data-ttu-id="24cb5-110">Skriv 'Udføre Contoso-projektet' i feltet Navn.</span><span class="sxs-lookup"><span data-stu-id="24cb5-110">In the Name field, type 'Complete the Contoso project'.</span></span>

    * <span data-ttu-id="24cb5-112">Mål kan vedrøre en enkeltperson, en gruppe eller endda en virksomhed.</span><span class="sxs-lookup"><span data-stu-id="24cb5-112">Goals can apply to an individual, a team, or even a corporation.</span></span> <span data-ttu-id="24cb5-113">Standardindstillingen for Medarbejdermål er Individuelt.</span><span class="sxs-lookup"><span data-stu-id="24cb5-113">The default setting for Employee goals is Individual.</span></span>  
6. <span data-ttu-id="24cb5-114">Indtast eller vælg en værdi i feltet Målkategori.</span><span class="sxs-lookup"><span data-stu-id="24cb5-114">In the Goal category field, enter or select a value.</span></span>
7. <span data-ttu-id="24cb5-115">Skift den dato, der vises i feltet Startdato.</span><span class="sxs-lookup"><span data-stu-id="24cb5-115">In the Start date field, change the date shown.</span></span>
8. <span data-ttu-id="24cb5-116">Skift den dato, der vises i feltet Slutdato.</span><span class="sxs-lookup"><span data-stu-id="24cb5-116">In the End date field, change the date shown.</span></span>
    * <span data-ttu-id="24cb5-117">Når du fuldfører et mål, skal du angive den dato, du fuldførte mål, her.</span><span class="sxs-lookup"><span data-stu-id="24cb5-117">When you complete a goal, enter the date that you completed the goal here.</span></span>  
9. <span data-ttu-id="24cb5-118">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="24cb5-118">Click Save.</span></span>
10. <span data-ttu-id="24cb5-119">Udvid afsnittet Målinger.</span><span class="sxs-lookup"><span data-stu-id="24cb5-119">Expand the Measurements section.</span></span> <span data-ttu-id="24cb5-120">Hvis det allerede er udvidet, skal du klikke på pilen Udvid to gange.</span><span class="sxs-lookup"><span data-stu-id="24cb5-120">If it is already expanded, click the Expand arrow twice.</span></span>
11. <span data-ttu-id="24cb5-121">Klik på Tilføj mål for at åbne dialogboksen.</span><span class="sxs-lookup"><span data-stu-id="24cb5-121">Click Add measurement to open the drop dialog.</span></span>
12. <span data-ttu-id="24cb5-122">Indtast eller vælg en værdi i feltet Målinger.</span><span class="sxs-lookup"><span data-stu-id="24cb5-122">In the Measurement field, enter or select a value.</span></span>
13. <span data-ttu-id="24cb5-123">Angiv et tal i feltet Målbeløb.</span><span class="sxs-lookup"><span data-stu-id="24cb5-123">In the Target amount field, enter a number.</span></span>
14. <span data-ttu-id="24cb5-124">Klik på OK.</span><span class="sxs-lookup"><span data-stu-id="24cb5-124">Click OK.</span></span>
15. <span data-ttu-id="24cb5-125">Udvid afsnittet Aktiviteter.</span><span class="sxs-lookup"><span data-stu-id="24cb5-125">Expand the Activities section.</span></span> <span data-ttu-id="24cb5-126">Hvis det allerede er udvidet, skal du klikke på pilen Udvid to gange.</span><span class="sxs-lookup"><span data-stu-id="24cb5-126">If it is already expanded, click the Expand arrow twice.</span></span>
16. <span data-ttu-id="24cb5-127">Klik på Tilføj.</span><span class="sxs-lookup"><span data-stu-id="24cb5-127">Click Add.</span></span>
17. <span data-ttu-id="24cb5-128">Skriv en værdi i feltet Titel.</span><span class="sxs-lookup"><span data-stu-id="24cb5-128">In the Title field, type a value.</span></span>
18. <span data-ttu-id="24cb5-129">Skriv en værdi i feltet Beskrivelse.</span><span class="sxs-lookup"><span data-stu-id="24cb5-129">In the Description field, type a value.</span></span>
19. <span data-ttu-id="24cb5-130">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="24cb5-130">Click Save.</span></span>
20. <span data-ttu-id="24cb5-131">Klik på Vis sektioner for at åbne dialogboksen.</span><span class="sxs-lookup"><span data-stu-id="24cb5-131">Click Show sections to open the drop dialog.</span></span>
21. <span data-ttu-id="24cb5-132">Vælg Nej i feltet Vis målinger.</span><span class="sxs-lookup"><span data-stu-id="24cb5-132">Select No in the Show measurements field.</span></span>
22. <span data-ttu-id="24cb5-133">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="24cb5-133">Click Save.</span></span>
23. <span data-ttu-id="24cb5-134">Klik på Vis sektioner for at åbne dialogboksen.</span><span class="sxs-lookup"><span data-stu-id="24cb5-134">Click Show sections to open the drop dialog.</span></span>
24. <span data-ttu-id="24cb5-135">Vælg Ja i feltet Vis målinger.</span><span class="sxs-lookup"><span data-stu-id="24cb5-135">Select Yes in the Show measurements field.</span></span>
25. <span data-ttu-id="24cb5-136">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="24cb5-136">Click Save.</span></span>
    * <span data-ttu-id="24cb5-137">Der er ofte mål, der er de samme eller ens for gruppemedlemmer eller alle medarbejdere.</span><span class="sxs-lookup"><span data-stu-id="24cb5-137">There are often goals that are the same or similar for team members or all employees.</span></span>     <span data-ttu-id="24cb5-138">Personaleadministratoren eller lederen kan oprette målskabeloner for disse mål.</span><span class="sxs-lookup"><span data-stu-id="24cb5-138">Your HR administrator or manager can create goal templates for those goals.</span></span> <span data-ttu-id="24cb5-139">Du kan tilføje et mål fra en skabelon og derefter redigere den, så den stemmer overens med dine tidslinjer.</span><span class="sxs-lookup"><span data-stu-id="24cb5-139">You can add a goal from a template, and then modify it to match your timelines.</span></span>
