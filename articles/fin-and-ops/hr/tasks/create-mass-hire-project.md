2. <span data-ttu-id="112bd-109">Skriv en værdi i feltet Masseansættelsesprojekt.</span><span class="sxs-lookup"><span data-stu-id="112bd-109">In the Mass hire project field, type a value.</span></span>

7. <span data-ttu-id="112bd-114">Klik på Ja.</span><span class="sxs-lookup"><span data-stu-id="112bd-114">Click Yes.</span></span>
8. <span data-ttu-id="112bd-115">Klik på Opret stillinger.</span><span class="sxs-lookup"><span data-stu-id="112bd-115">Click Create positions.</span></span>
