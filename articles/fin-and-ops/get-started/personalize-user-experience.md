<span data-ttu-id="6d725-243">Indstillingerne for dimensionsvisning gælder på tværs af alle sider, og disse indstillinger tilsidesætter den tilpassede opsætning af lagerdimensionsfelter på individuelle sider.</span><span class="sxs-lookup"><span data-stu-id="6d725-243">The Dimension display settings apply across all pages and these settings will override any personalized setup of inventory dimension fields on individual pages.</span></span> 

<span data-ttu-id="6d725-244">I eksemplet med batchnummer-lagerdimensionen, skal denne dimension fjernes som en del af indstillingen **Vis dimensioner** for tabellen, hvis denne kolonne ikke skal vises.</span><span class="sxs-lookup"><span data-stu-id="6d725-244">For the example with the Batch number inventory dimension, this dimension would have to be cleared as part of the **Display dimensions** option for the table to not display this column.</span></span> <span data-ttu-id="6d725-245">Denne ændring vil desuden gælde ikke kun for en bestemt side, men på tværs af alle sider.</span><span class="sxs-lookup"><span data-stu-id="6d725-245">Eventually this change would apply not only on one specific page but across all pages.</span></span>
