[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="f52a6-104">En driftsenhed er en organisation, der bruges til at opdele administrationen af økonomiske ressourcer og driftsprocesser i et firma.</span><span class="sxs-lookup"><span data-stu-id="f52a6-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="f52a6-105">Personer i en driftsenhed er forpligtet til at maksimere brugen af knappe ressourcer, til at forbedre processer og til at redegøre for deres præstation.</span><span class="sxs-lookup"><span data-stu-id="f52a6-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="f52a6-106">Typerne af driftsenheder inkluderer bærere, virksomhedsenheder, afdelinger og værdistrømme.</span><span class="sxs-lookup"><span data-stu-id="f52a6-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="f52a6-107">Du kan bruge følgende procedure for at oprette en driftsenhed.</span><span class="sxs-lookup"><span data-stu-id="f52a6-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="f52a6-108">Det demodatafirma, der bruges til at oprette denne procedure, er USMF.</span><span class="sxs-lookup"><span data-stu-id="f52a6-108">The demo data company used to create this procedure is USMF.</span></span>

3. <span data-ttu-id="f52a6-111">Find og vælg den ønskede post på listen.</span><span class="sxs-lookup"><span data-stu-id="f52a6-111">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="f52a6-112">Vælg den type driftsenhed, du vil oprette.</span><span class="sxs-lookup"><span data-stu-id="f52a6-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="f52a6-113">Klik op linket i den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="f52a6-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="f52a6-114">Skriv en værdi i feltet Navn.</span><span class="sxs-lookup"><span data-stu-id="f52a6-114">In the Name field, type a value.</span></span>
    * <span data-ttu-id="f52a6-115">Udvid sektionen Generelt, hvis det er nødvendigt.</span><span class="sxs-lookup"><span data-stu-id="f52a6-115">Expand the General section, if necessary.</span></span>
