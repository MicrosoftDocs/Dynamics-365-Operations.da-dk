# <a name="configure-the-global-address-book"></a><span data-ttu-id="8b335-103">Konfiguration af den globale adressebog</span><span class="sxs-lookup"><span data-stu-id="8b335-103">Configure the global address book</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8b335-104">Brug denne procedure til at angive standardværdier og sikkerhedspolitikker for det globale adressekartotek.</span><span class="sxs-lookup"><span data-stu-id="8b335-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="8b335-105">Det demodatafirma, der bruges til at oprette denne opgave, er USMF.</span><span class="sxs-lookup"><span data-stu-id="8b335-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="8b335-106">Denne opgave er beregnet til planlægnings- og konfigurationsteamet.</span><span class="sxs-lookup"><span data-stu-id="8b335-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="8b335-107">Gå i navigationsruden til **Moduler > Organisationsadministration > Globalt adressekartotek > Parametre for globalt adressekartotek**.</span><span class="sxs-lookup"><span data-stu-id="8b335-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="8b335-108">Vælg, hvordan navne skal vises, i feltet **Navnerækkefølge**.</span><span class="sxs-lookup"><span data-stu-id="8b335-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="8b335-109">I **Slet parter uden roller**skal du vælge, om du vil slette parter, der ikke har fået tildelt en rolle.</span><span class="sxs-lookup"><span data-stu-id="8b335-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="8b335-110">I **Anvend dobbeltkontrol** skal du vælge, om du vil kontrollere for dublerede poster.</span><span class="sxs-lookup"><span data-stu-id="8b335-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="8b335-111">I **Vis DUNS-nummer på adresser** skal du vælge, om DUNS-nummeret skal vises på adresser.</span><span class="sxs-lookup"><span data-stu-id="8b335-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="8b335-112">I **Kontrollér for entydigt DUNS-nummer** skal du vælge, om du vil kontrollere for entydige DUNS-numre.</span><span class="sxs-lookup"><span data-stu-id="8b335-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="8b335-113">Vælg en indstilling i feltet **Part**.</span><span class="sxs-lookup"><span data-stu-id="8b335-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="8b335-114">Vælg en indstilling i feltet **Kunde**.</span><span class="sxs-lookup"><span data-stu-id="8b335-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="8b335-115">Vælg en indstilling i feltet **Leverandør**.</span><span class="sxs-lookup"><span data-stu-id="8b335-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="8b335-116">Vælg en indstilling i feltet **Kundeemne**.</span><span class="sxs-lookup"><span data-stu-id="8b335-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="8b335-117">Vælg en indstilling i feltet **Konkurrent**.</span><span class="sxs-lookup"><span data-stu-id="8b335-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="8b335-118">Klik på fanen **Sikkerhed for privat lokation**.</span><span class="sxs-lookup"><span data-stu-id="8b335-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="8b335-119">Find og vælg den ønskede post på listen.</span><span class="sxs-lookup"><span data-stu-id="8b335-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="8b335-120">Tryk på tasten Skift for at vælge flere roller, der skal føjes til ruden **Valgte roller**, og klik derefter på pilen for at tilføje de valgte roller.</span><span class="sxs-lookup"><span data-stu-id="8b335-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="8b335-121">Klik på **Gem**.</span><span class="sxs-lookup"><span data-stu-id="8b335-121">Click **Save**.</span></span>
