6. <span data-ttu-id="fb9a0-115">Indtast en værdi i feltet **Beskrivelse**.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-115">In the **Description** field, type a value.</span></span> <span data-ttu-id="fb9a0-116">Valutaen for gebyret er som standard debitorvalutaen.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-116">The currency for the fee defaults to the customer currency.</span></span> <span data-ttu-id="fb9a0-117">Valutakoden kan være forskellig fra fakturaens valuta.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-117">This currency code can be different than the invoice currency.</span></span>

8. <span data-ttu-id="fb9a0-121">I feltet til rykkerkoden skal du vælge den næste rykker, der skal sendes i rækkefølgen.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-121">In the collection letter code field, select the next collection letter that will be sent in the sequence.</span></span>
9. <span data-ttu-id="fb9a0-122">Indtast en værdi i feltet **Beskrivelse**.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-122">In the **Description** field, type a value.</span></span>
10. <span data-ttu-id="fb9a0-123">I feltet **Hovedkonto** skal du vælge den omsætningskonto, der skal bruges til gebyrer.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-123">In the **Main account** field, select the revenue account that will be used for fees.</span></span>

13. <span data-ttu-id="fb9a0-127">Klik op linket i den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-127">In the list, click the link in the selected row.</span></span>

15. <span data-ttu-id="fb9a0-129">Angiv antallet af respitdage, som du vil tillade, i feltet **Dage**.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-129">In the **Days** field, enter the number of grace days that you will allow.</span></span> <span data-ttu-id="fb9a0-130">Dette er antallet af dage efter forfaldsdatoen, hvor der kan oprettes en rykker.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-130">This is the number of days after the due date that a collection letter can be generated.</span></span> <span data-ttu-id="fb9a0-131">Forfaldsdatoen, der bruges til beregningen, afhænger af placeringen af rykkeren i rykkersekvensen:</span><span class="sxs-lookup"><span data-stu-id="fb9a0-131">The due date that is used for the calculation depends on the position of the collection letter in the collection letter sequence:</span></span>

18. <span data-ttu-id="fb9a0-137">Indtast en værdi i feltet **Beskrivelse**.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-137">In the **Description** field, type a value.</span></span>
19. <span data-ttu-id="fb9a0-138">Angive de ønskede værdier i feltet **Hovedkonto**.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-138">In the **Main account** field, specify the desired values.</span></span>

22. <span data-ttu-id="fb9a0-141">Klik op linket i den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-141">In the list, click the link in the selected row.</span></span>
23. <span data-ttu-id="fb9a0-142">Skriv et tal i feltet **Mindste forsinkede saldo**.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-142">In the **Minimum overdue balance** field, enter a number.</span></span>
24. <span data-ttu-id="fb9a0-143">Angiv et tal i feltet **Dage**.</span><span class="sxs-lookup"><span data-stu-id="fb9a0-143">In the **Days** field, enter a number.</span></span>
