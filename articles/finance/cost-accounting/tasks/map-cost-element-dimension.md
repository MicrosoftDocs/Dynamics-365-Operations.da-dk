# <a name="map-a-cost-element-dimension"></a><span data-ttu-id="25d84-103">Tilknytte en dimension for omkostningselement</span><span class="sxs-lookup"><span data-stu-id="25d84-103">Map a cost element dimension</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="25d84-104">En omkostningscontroller kan bruge denne procedure til at knytte en dimension for et omkostningselement til en dimension for et omkostningselement i den juridiske enhed for MXMF.</span><span class="sxs-lookup"><span data-stu-id="25d84-104">A cost controller can use this procedure to map a cost element dimension to a cost element dimension in the MXMF legal entity.</span></span> <span data-ttu-id="25d84-105">Denne registrering bruger USP2-demodatafirmaet.</span><span class="sxs-lookup"><span data-stu-id="25d84-105">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="25d84-106">Gå til Omkostningsregnskab > Dimensioner > Dimensioner for omkostningselement.</span><span class="sxs-lookup"><span data-stu-id="25d84-106">Go to Cost accounting > Dimensions > Cost element dimensions.</span></span>
2. <span data-ttu-id="25d84-107">Find og vælg den ønskede post på listen.</span><span class="sxs-lookup"><span data-stu-id="25d84-107">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="25d84-108">Vælg Omkostningselementer i dette eksempel.</span><span class="sxs-lookup"><span data-stu-id="25d84-108">For this example, select Cost elements.</span></span>  
3. <span data-ttu-id="25d84-109">Klik på Tilknytninger af dimensioner.</span><span class="sxs-lookup"><span data-stu-id="25d84-109">Click Dimension mappings.</span></span>
4. <span data-ttu-id="25d84-110">Klik på Konfigurer tilknytninger fra denne dimension.</span><span class="sxs-lookup"><span data-stu-id="25d84-110">Click Configure mappings from this dimension.</span></span>
5. <span data-ttu-id="25d84-111">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="25d84-111">Click New.</span></span>
6. <span data-ttu-id="25d84-112">Indtast eller vælg en værdi i feltet Til dimension.</span><span class="sxs-lookup"><span data-stu-id="25d84-112">In the To dimension field, enter or select a value.</span></span>
    * <span data-ttu-id="25d84-113">Vælg MXMF-omkostningselementer i dette eksempel.</span><span class="sxs-lookup"><span data-stu-id="25d84-113">For this example, select MXMF Cost elements.</span></span>  
7. <span data-ttu-id="25d84-114">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="25d84-114">Click New.</span></span>
8. <span data-ttu-id="25d84-115">Markér den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="25d84-115">In the list, mark the selected row.</span></span>
9. <span data-ttu-id="25d84-116">Indtast eller vælg en værdi i feltet Fra dimensionsmedlem.</span><span class="sxs-lookup"><span data-stu-id="25d84-116">In the From dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="25d84-117">I dette eksempel skal du vælge dimensionsmedlem 606400 Udgift til telefon og fax.</span><span class="sxs-lookup"><span data-stu-id="25d84-117">For this example, select dimension member 606400 Telephone & Fax Expense.</span></span>  
10. <span data-ttu-id="25d84-118">Indtast eller vælg en værdi i feltet Til dimensionsmedlem.</span><span class="sxs-lookup"><span data-stu-id="25d84-118">In the To dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="25d84-119">I dette eksempel skal du vælge dimensionsmedlem 6001004 Telefono.</span><span class="sxs-lookup"><span data-stu-id="25d84-119">For this example, select dimension member 6001004 Telefono.</span></span>  
11. <span data-ttu-id="25d84-120">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="25d84-120">Click Save.</span></span>
