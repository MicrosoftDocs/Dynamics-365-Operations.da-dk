# <a name="configure-cost-control-workspace-parameters"></a><span data-ttu-id="54ced-103">Konfigurere arbejdsområdeparametre for omkostningsstyring</span><span class="sxs-lookup"><span data-stu-id="54ced-103">Configure cost control workspace parameters</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="54ced-104">Du kan bruge denne procedure til at konfigurere arbejdsområdet Omkostningsstyring, så ledere på forskellige niveauer i organisationen kan få indsigt i deres omkostningsobjekter, f.eks. bærere og produktgrupper.</span><span class="sxs-lookup"><span data-stu-id="54ced-104">Use this procedure to configure the Cost control workspace so that managers at various levels in an organization can gain insight into their cost objects, such as cost centers and product groups.</span></span> <span data-ttu-id="54ced-105">USP2-demofirmaet blev brugt til at oprette denne registrering.</span><span class="sxs-lookup"><span data-stu-id="54ced-105">The USP2 demo company was used to create this recording.</span></span>

1. <span data-ttu-id="54ced-106">Gå til Omkostningsregnskab > Opsætning > Konfiguration af arbejdsområde for omkostningsstyring.</span><span class="sxs-lookup"><span data-stu-id="54ced-106">Go to Cost accounting > Setup > Cost control workspace configuration.</span></span>
2. <span data-ttu-id="54ced-107">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="54ced-107">Click New.</span></span>
3. <span data-ttu-id="54ced-108">Skriv en værdi i feltet Navn.</span><span class="sxs-lookup"><span data-stu-id="54ced-108">In the Name field, type a value.</span></span>
4. <span data-ttu-id="54ced-109">Skriv en værdi i feltet Beskrivelse.</span><span class="sxs-lookup"><span data-stu-id="54ced-109">In the Description field, type a value.</span></span>
5. <span data-ttu-id="54ced-110">Vælg Ja i feltet Publiceret.</span><span class="sxs-lookup"><span data-stu-id="54ced-110">Select Yes in the Published field.</span></span>
    * <span data-ttu-id="54ced-111">Hvis du vælger Ja for denne indstilling, kan brugere, der er tildelt en af disse roller, se rapporten i arbejdsområdet Omkostningsstyring: regnskabschef, bogholder, bogholderimedarbejder eller omkostningsobjektcontroller.</span><span class="sxs-lookup"><span data-stu-id="54ced-111">If you set this option to Yes, users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, cost accountant clerk, or cost object controller.</span></span> <span data-ttu-id="54ced-112">Hvis du vælger Nej for denne indstilling, er det kun brugere, der er tildelt en af disse roller, der kan se rapporten i arbejdsområdet Omkostningsstyring: regnskabschef, bogholder eller bogholderimedarbejder.</span><span class="sxs-lookup"><span data-stu-id="54ced-112">If you set this option to No, only users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, or cost accountant clerk.</span></span>  
6. <span data-ttu-id="54ced-113">Udvid datafiltreringssektionen.</span><span class="sxs-lookup"><span data-stu-id="54ced-113">Expand the Data filtering section.</span></span>
7. <span data-ttu-id="54ced-114">Indtast eller vælg en værdi i feltet Omkostningskontrolenhed.</span><span class="sxs-lookup"><span data-stu-id="54ced-114">In the Cost control unit field, enter or select a value.</span></span>
8. <span data-ttu-id="54ced-115">Skriv eller vælg en værdi i feltet Oprindelig budgetversion.</span><span class="sxs-lookup"><span data-stu-id="54ced-115">In the Budget original version field, enter or select a value.</span></span>
9. <span data-ttu-id="54ced-116">Indtast eller vælg en værdi i feltet Dimensionshierarki for omkostningselement.</span><span class="sxs-lookup"><span data-stu-id="54ced-116">In the Cost element dimension hierarchy field, enter or select a value.</span></span>
10. <span data-ttu-id="54ced-117">Indtast eller vælg en værdi i feltet Dimensionshierarki for omkostningsobjekt.</span><span class="sxs-lookup"><span data-stu-id="54ced-117">In the Cost object dimension hierarchy field, enter or select a value.</span></span>
11. <span data-ttu-id="54ced-118">Udvid sektionen Tildel beregningsposter.</span><span class="sxs-lookup"><span data-stu-id="54ced-118">Expand the Assign calculation records section.</span></span>
12. <span data-ttu-id="54ced-119">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="54ced-119">Click New.</span></span>
13. <span data-ttu-id="54ced-120">Markér den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="54ced-120">In the list, mark the selected row.</span></span>
14. <span data-ttu-id="54ced-121">Indtast eller vælg en værdi i feltet Regnskabskalenderperiode.</span><span class="sxs-lookup"><span data-stu-id="54ced-121">In the Fiscal calendar period field, enter or select a value.</span></span>
15. <span data-ttu-id="54ced-122">Indtast eller vælg en værdi i feltet Aktuel version.</span><span class="sxs-lookup"><span data-stu-id="54ced-122">In the Actual version field, enter or select a value.</span></span>
16. <span data-ttu-id="54ced-123">Udvid sektionen Regnskabsperioder pr. kolonne.</span><span class="sxs-lookup"><span data-stu-id="54ced-123">Expand the Fiscal periods per column section.</span></span>
17. <span data-ttu-id="54ced-124">Vælg Ja i feltet Aktuel periode.</span><span class="sxs-lookup"><span data-stu-id="54ced-124">Select Yes in the Current period field.</span></span>
18. <span data-ttu-id="54ced-125">Udvid sektionen Kolonner, der skal vises for omkostninger.</span><span class="sxs-lookup"><span data-stu-id="54ced-125">Expand the Columns to display for costs section.</span></span>
19. <span data-ttu-id="54ced-126">Vælg Ja i feltet Anlægsaktiv.</span><span class="sxs-lookup"><span data-stu-id="54ced-126">Select Yes in the Fixed cost field.</span></span>
20. <span data-ttu-id="54ced-127">Vælg Ja i feltet Variabel omkostning.</span><span class="sxs-lookup"><span data-stu-id="54ced-127">Select Yes in the Variable cost field.</span></span>
21. <span data-ttu-id="54ced-128">Vælg Ja i feltet Totalomkostning.</span><span class="sxs-lookup"><span data-stu-id="54ced-128">Select Yes in the Total cost field.</span></span>
22. <span data-ttu-id="54ced-129">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="54ced-129">Click Save.</span></span>
23. <span data-ttu-id="54ced-130">Luk siden.</span><span class="sxs-lookup"><span data-stu-id="54ced-130">Close the page.</span></span>
24. <span data-ttu-id="54ced-131">Gå til Omkostningsregnskab > Arbejdsområder > Omkostningsstyring.</span><span class="sxs-lookup"><span data-stu-id="54ced-131">Go to Cost accounting > Workspaces > Cost control.</span></span>
25. <span data-ttu-id="54ced-132">Vælg en opgørelse for at få vist faste, variable, samlede og ikke-klassificerede omkostninger for de valgte regnskabsperioder.</span><span class="sxs-lookup"><span data-stu-id="54ced-132">Select a statement to see fixed, variable, total, and unclassified costs for the selected fiscal periods.</span></span>
26. <span data-ttu-id="54ced-133">Indtast eller vælg en værdi i feltet Regnskabskalenderperiode.</span><span class="sxs-lookup"><span data-stu-id="54ced-133">In the Fiscal calendar period field, enter or select a value.</span></span>
27. <span data-ttu-id="54ced-134">Indtast eller vælg en værdi i feltet Dimensionshierarkinode for omkostningsobjekt.</span><span class="sxs-lookup"><span data-stu-id="54ced-134">In the Cost object dimension hierarchy node field, enter or select a value.</span></span>
    * <span data-ttu-id="54ced-135">Når du har valgt et dimensionshierarki for omkostningsobjekt, kan du udvide dimensionshierarkiet for omkostningselement for at få vist de ønskede omkostningsværdier.</span><span class="sxs-lookup"><span data-stu-id="54ced-135">After you've selected a Cost object dimension hierarchy, expand the Cost element dimension hierarchy to see the desired cost values.</span></span> <span data-ttu-id="54ced-136">For eksempel kan du udvide hierarkiet for at få vist værdien i Produktionsomkostninger.</span><span class="sxs-lookup"><span data-stu-id="54ced-136">For example, you can expand the hierarchy to Manufacturing overhead to see the value.</span></span>
