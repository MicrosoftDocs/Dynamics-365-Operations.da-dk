# <a name="create-a-cost-accounting-ledger"></a><span data-ttu-id="6b368-103">Oprette en finanspost for driftsregnskab</span><span class="sxs-lookup"><span data-stu-id="6b368-103">Create a cost accounting ledger</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="6b368-104">En finanspost for omkostningsregnskab repræsenterer den overordnede rapporteringsenhed.</span><span class="sxs-lookup"><span data-stu-id="6b368-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="6b368-105">Den er defineret af en dimension for omkostningselement, statistisk dimension, regnskabskalender og valuta.</span><span class="sxs-lookup"><span data-stu-id="6b368-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="6b368-106">Det er en uafhængig af begrebet juridiske enheder.</span><span class="sxs-lookup"><span data-stu-id="6b368-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="6b368-107">En juridisk enhed og data, der kan knyttes til mange finansposter i omkostningsregnskab.</span><span class="sxs-lookup"><span data-stu-id="6b368-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="6b368-108">Denne registrering bruger USP2-demodatafirmaet.</span><span class="sxs-lookup"><span data-stu-id="6b368-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="6b368-109">Gå til Omkostningsregnskab > Opsætning Finans > Finansposter for omkostningsregnskab.</span><span class="sxs-lookup"><span data-stu-id="6b368-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="6b368-110">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="6b368-110">Click New.</span></span>
3. <span data-ttu-id="6b368-111">Skriv en værdi i feltet Navn.</span><span class="sxs-lookup"><span data-stu-id="6b368-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="6b368-112">Indtast eller vælg en værdi i feltet Dimension for omkostningselement.</span><span class="sxs-lookup"><span data-stu-id="6b368-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="6b368-113">Indtast eller vælg en værdi i feltet Regnskabskalender.</span><span class="sxs-lookup"><span data-stu-id="6b368-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="6b368-114">Skriv eller vælg en værdi i feltet Regnskabsvaluta.</span><span class="sxs-lookup"><span data-stu-id="6b368-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="6b368-115">Indtast eller vælg en værdi i feltet Valutakurstype.</span><span class="sxs-lookup"><span data-stu-id="6b368-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="6b368-116">Indtast eller vælg en værdi i feltet Statistisk dimension.</span><span class="sxs-lookup"><span data-stu-id="6b368-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="6b368-117">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="6b368-117">Click Save.</span></span>
