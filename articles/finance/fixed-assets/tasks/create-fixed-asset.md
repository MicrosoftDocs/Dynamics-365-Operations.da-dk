[!include [banner](../../includes/banner.md)]

<span data-ttu-id="3dc23-104">Denne opgaveguide anvender demofirmaet USMF.</span><span class="sxs-lookup"><span data-stu-id="3dc23-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="3dc23-105">Det opretter et nyt anlægsaktiv ved hjælp af listesiden Anlægsaktiv.</span><span class="sxs-lookup"><span data-stu-id="3dc23-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="3dc23-106">Gå til **Navigationsrude > Moduler > Anlægsaktiver > Anlægsaktiver > Anlægsaktiver**.</span><span class="sxs-lookup"><span data-stu-id="3dc23-106">Go to **Navigation pane > Modules > Fixed assets > Fixed assets > Fixed assets**.</span></span>
2. <span data-ttu-id="3dc23-107">Klik på **Ny** i **handlingsruden**.</span><span class="sxs-lookup"><span data-stu-id="3dc23-107">On the **Action pane**, click **New**.</span></span>

5. <span data-ttu-id="3dc23-113">Klik på **Bøger** i **handlingsruden**.</span><span class="sxs-lookup"><span data-stu-id="3dc23-113">On the **Action pane**, click **Books**.</span></span>
