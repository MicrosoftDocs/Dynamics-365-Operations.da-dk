# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="8d39a-103">Fjerne et anlægsaktiv ved hjælp af en fritekstfaktura</span><span class="sxs-lookup"><span data-stu-id="8d39a-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8d39a-104">Dette emne viser, hvordan du kasserer et anlægsaktiv ved hjælp af fritekstfakturaen.</span><span class="sxs-lookup"><span data-stu-id="8d39a-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="8d39a-105">I navigationsruden skal du gå til **Moduler > Debitor > Fakturaer > Alle fritekstfakturaer**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="8d39a-106">Vælg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-106">Select **New**.</span></span>
3. <span data-ttu-id="8d39a-107">Indtast eller vælg en værdi i feltet **Debitorkonto**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="8d39a-108">Kontrollér standarddatoen for **Faktura**, og rediger den efter behov.</span><span class="sxs-lookup"><span data-stu-id="8d39a-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="8d39a-109">Kontrollér resterende standardoverskriftsfelter som f.eks. **Valuta**, og rediger efter behov.</span><span class="sxs-lookup"><span data-stu-id="8d39a-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="8d39a-110">Kontrollér felterne **Beskrivelse** og **Hovedkonto** i afsnittet **Fakturalinjer**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="8d39a-111">Kontrollér standardværdien i felterne **Momsgruppe** og **Varemomsgruppe**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="8d39a-112">Angiv **Enhedspris** eller **Beløb** for salg af anlægsaktivet.</span><span class="sxs-lookup"><span data-stu-id="8d39a-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="8d39a-113">Vælg sektionen **Linjedetaljer**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="8d39a-114">Skriv eller vælg en værdi i feltet **Anlægsaktiv**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="8d39a-115">Vælg **Bogfør**.</span><span class="sxs-lookup"><span data-stu-id="8d39a-115">Select **Post**.</span></span>
