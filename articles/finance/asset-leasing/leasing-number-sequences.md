[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="1272a-105">Dette emne forklarer, hvordan du opretter nummerserier til leasingaftale-id'er.</span><span class="sxs-lookup"><span data-stu-id="1272a-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="1272a-106">Det forklarer også, hvordan du opretter entydige id'er, der bruges i indeksværdireguleringsprocessen.</span><span class="sxs-lookup"><span data-stu-id="1272a-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>

1. <span data-ttu-id="1272a-107">Gå til **Aktivleasing \> Opsætning \> Parametre til aktivleasing**.</span><span class="sxs-lookup"><span data-stu-id="1272a-107">Go to **Asset leasing \> Setup \> Asset leasing parameters**.</span></span>

6. <span data-ttu-id="1272a-114">Vælg nummerserien for **Id for fratrædelsesforslag**-referencen.</span><span class="sxs-lookup"><span data-stu-id="1272a-114">Select the number sequence for the **Termination Proposal ID** reference.</span></span> <span data-ttu-id="1272a-115">Denne nummerserie vil blive brugt til at generere det entydige id for hver fratrædelsesforslag.</span><span class="sxs-lookup"><span data-stu-id="1272a-115">This number sequence will be used to generate the unique identifier for each termination proposal.</span></span>
