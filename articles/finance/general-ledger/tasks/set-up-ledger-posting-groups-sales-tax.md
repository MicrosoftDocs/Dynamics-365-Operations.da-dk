# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="f0017-103">Oprette finanskonteringsgrupper til moms</span><span class="sxs-lookup"><span data-stu-id="f0017-103">Set up Ledger posting groups for sales tax</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f0017-104">Momsen beregnes og bogføres på hovedkonti, der er angivet i finanskonteringsgrupperne.</span><span class="sxs-lookup"><span data-stu-id="f0017-104">Sales tax is calculated and posted to main accounts that are specified in the Ledger posting groups.</span></span> <span data-ttu-id="f0017-105">Finanskonteringsgrupperne er tilknyttet de enkelte momskoder.</span><span class="sxs-lookup"><span data-stu-id="f0017-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="f0017-106">Du kan oprette individuelle finanskonteringsgrupper til hver enkelt momskode, bruge én finanskonteringsgruppe til alle momskoder eller knytte flere finanskonteringsgrupper til momskoderne.</span><span class="sxs-lookup"><span data-stu-id="f0017-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="f0017-107">Denne registrering anvender demofirmaet DEMF.</span><span class="sxs-lookup"><span data-stu-id="f0017-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="f0017-108">Gå til **Navigationsrude > Moduler > Moms > Opsætning > Moms > Finanskonteringsgrupper**.</span><span class="sxs-lookup"><span data-stu-id="f0017-108">Go to **Navigation pane > Modules > Tax > Setup > Sales tax > Ledger posting groups**.</span></span>
2. <span data-ttu-id="f0017-109">Klik på **Ny**.</span><span class="sxs-lookup"><span data-stu-id="f0017-109">Click **New**.</span></span>
3. <span data-ttu-id="f0017-110">Indtast en værdi i feltet **Finanskonteringsgruppe**.</span><span class="sxs-lookup"><span data-stu-id="f0017-110">In the **Ledger posting group** field, type a value.</span></span>
4. <span data-ttu-id="f0017-111">Indtast en værdi i feltet **Beskrivelse**.</span><span class="sxs-lookup"><span data-stu-id="f0017-111">In the **Description** field, type a value.</span></span>
5. <span data-ttu-id="f0017-112">Vælg hovedkontoen til udgående moms, som skal betales til skattemyndighederne, i feltet **Udgående moms**.</span><span class="sxs-lookup"><span data-stu-id="f0017-112">In the **Sales tax payable** field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span> <span data-ttu-id="f0017-113">Moms opkræves på vegne af skattemyndighederne, når du sælger afgiftspligtige varer og tjenesteydelser.</span><span class="sxs-lookup"><span data-stu-id="f0017-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="f0017-114">Vælg hovedkontoen til indgående moms, som skal modtages fra skattemyndighederne, i feltet **Indgående moms**.</span><span class="sxs-lookup"><span data-stu-id="f0017-114">In the **Sales tax receivable** field, select the main account for incoming taxes that are received from the tax authority.</span></span> <span data-ttu-id="f0017-115">Kreditorer opkræver moms på vegne af skattemyndighederne, når du køber afgiftspligtige varer og tjenesteydelser.</span><span class="sxs-lookup"><span data-stu-id="f0017-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="f0017-116">Dette felt er ikke tilgængeligt, hvis indstillingen Anvend momsregler er valgt på siden **Finansparametre**.</span><span class="sxs-lookup"><span data-stu-id="f0017-116">This field is not available if the Apply sales tax taxation rules option is selected in the **General ledger parameters** page.</span></span> <span data-ttu-id="f0017-117">Den moms, der betales til kreditorer, debiteres i stedet samme konto som købet.</span><span class="sxs-lookup"><span data-stu-id="f0017-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="f0017-118">Vælg i feltet **Udgift for importmoms** den hovedkonto til bogføring af fradragsberettiget importmoms, der ikke er opkrævet eller rapporteret til skattemyndighederne af kreditorer som en del af EU-modtagermoms GST/HST.</span><span class="sxs-lookup"><span data-stu-id="f0017-118">In the **Use tax expense** field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span> <span data-ttu-id="f0017-119">Indstillingen **Importmoms** skal være markeret for **Momskode** i den **Momsgruppe**, der bruges i transaktionen.</span><span class="sxs-lookup"><span data-stu-id="f0017-119">The **Use tax** option needs to be selected for the **Sales tax code** in the **Sales tax group** that is used in the transaction.</span></span> <span data-ttu-id="f0017-120">Dette felt er ikke tilgængeligt, hvis indstillingen **Anvend momsregler** er valgt på siden **Finansparametre**.</span><span class="sxs-lookup"><span data-stu-id="f0017-120">This field is not be available if the **Apply sales tax taxation rules** option is selected in the **General ledger parameters** page.</span></span>   
8. <span data-ttu-id="f0017-121">Vælg hovedkontoen til bogføring af indgående moms, som skal betales til skattemyndighederne, i feltet **Skyldig importmoms**.</span><span class="sxs-lookup"><span data-stu-id="f0017-121">In the **Use tax payable** field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span> <span data-ttu-id="f0017-122">Indstillingen **Importmoms** skal være markeret for **Momskode** i **Momsgruppe**, før **Importmoms** kan bogføres.</span><span class="sxs-lookup"><span data-stu-id="f0017-122">The **Use tax** option needs to be selected in the **Sales tax code** in the **Sales tax group** to post **Use tax**.</span></span> <span data-ttu-id="f0017-123">Hvis indstillingen **Anvend momsregler** er valgt på siden **Finansparametre**, bogføres modregningen til transaktionens udgiftskonto.</span><span class="sxs-lookup"><span data-stu-id="f0017-123">If **Apply sales tax taxation rules** option is selected in **General ledger parameters** page, the offset is posted to the transaction's expense account.</span></span>   
9. <span data-ttu-id="f0017-124">Vælg i feltet **Afregningskonto** den hovedkonto, som nettosaldoen for finanskontiene, der er angivet i felterne **Skyldig importmoms** og **Indgående moms**, skal bogføres på.</span><span class="sxs-lookup"><span data-stu-id="f0017-124">In the **Settlement account** field, select the main account that the net balance of the ledger accounts specified in the **Use tax payable** and **Sales tax receivable** fields will be posted.</span></span> <span data-ttu-id="f0017-125">Saldoen oprettes, når du afregner momsen, og bogføringsjobbet køres.</span><span class="sxs-lookup"><span data-stu-id="f0017-125">The balance will be created when the sales tax settle and post job is ran.</span></span>  <span data-ttu-id="f0017-126">Hvis momsmyndigheden for afregningsperioden er knyttet til en kreditorkonto, bogføres saldoen i stedet på kreditorens konto.</span><span class="sxs-lookup"><span data-stu-id="f0017-126">If the tax authority for the settlement period is associated with a vendor account, the balance is posted to the vendor account instead.</span></span>
10. <span data-ttu-id="f0017-127">Vælg i feltet **Kreditor kasserabat** den hovedkonto, der skal bogføres kasserabat på for de momskoder, der er tilknyttet denne finanskonteringsgruppe.</span><span class="sxs-lookup"><span data-stu-id="f0017-127">In the **Vendor cash discount** field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span> <span data-ttu-id="f0017-128">Det er valgfrit, og hvis der ikke angives, bruges den hovedkonto på **Kasserabatkoder**.</span><span class="sxs-lookup"><span data-stu-id="f0017-128">This is optional and if no account is entered,  the main account on **Cash discount codes** will be used.</span></span> <span data-ttu-id="f0017-129">Det kan være nyttigt at bruge forskellige konti pr. **Finanskonteringsgruppe**, hvis bruger tilbagefører momsen i kasserabatindstillingen på momsgrupper.</span><span class="sxs-lookup"><span data-stu-id="f0017-129">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="f0017-130">Vælg i feltet **Debitor, kasserabat** den hovedkonto, der skal bogføres kasserabat på for de **Momskoder**, der er tilknyttet denne **Finanskonteringsgruppe**.</span><span class="sxs-lookup"><span data-stu-id="f0017-130">In the **Customer case discount** field, select the main account to post cash discount for **Sales tax codes** associated with this **Ledger posting group**.</span></span> <span data-ttu-id="f0017-131">Det er valgfrit, og hvis der ikke angives, bruges den hovedkonto på **Kasserabatkoder**.</span><span class="sxs-lookup"><span data-stu-id="f0017-131">This is optional and if no account is entered, the main account on the **Cash discount codes** will be used.</span></span> <span data-ttu-id="f0017-132">Det kan være nyttigt at bruge forskellige konti pr. **Finanskonteringsgruppe**, hvis bruger tilbagefører momsen i kasserabatindstillingen på **Momsgrupper**.</span><span class="sxs-lookup"><span data-stu-id="f0017-132">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on **Sales tax groups**.</span></span>  
12. <span data-ttu-id="f0017-133">Klik på **Gem**.</span><span class="sxs-lookup"><span data-stu-id="f0017-133">Click **Save**.</span></span>
