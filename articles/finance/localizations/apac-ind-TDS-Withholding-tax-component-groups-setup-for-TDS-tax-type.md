# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="9c852-103">Konfigurere A-skat-komponentgrupper for TDS-skattetypen</span><span class="sxs-lookup"><span data-stu-id="9c852-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="9c852-104">Dette emne indeholder en forklaring på, hvordan du opretter komponentgrupper for A-skat, f.eks. **Leje** og **Kontrahent**, for skattetypen TDS, der fratrækkes ved kilden.</span><span class="sxs-lookup"><span data-stu-id="9c852-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="9c852-105">Gå til **Moms \> Opsætning \> A-skat \> A-skat-komponentgrupper**.</span><span class="sxs-lookup"><span data-stu-id="9c852-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="9c852-106">[![Siden A-skat-komponentgrupper](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="9c852-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="9c852-107">Vælg **TDS** i feltet **Skattetype** for at konfigurere komponentgrupper for TDS-skattetypen.</span><span class="sxs-lookup"><span data-stu-id="9c852-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="9c852-108">Vælg **Ny** i handlingsruden for at oprette en linje.</span><span class="sxs-lookup"><span data-stu-id="9c852-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="9c852-109">Angiv et navn for TDS-komponentgruppen i feltet **A-skat-komponentgrupper**.</span><span class="sxs-lookup"><span data-stu-id="9c852-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="9c852-110">Indtast en beskrivelse i feltet **Beskrivelse**.</span><span class="sxs-lookup"><span data-stu-id="9c852-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="9c852-111">I feltet **Status** i oversigtspanelet **Generelt** skal du vælge hjemmehørende-status for TDS-komponentgruppen.</span><span class="sxs-lookup"><span data-stu-id="9c852-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="9c852-112">Valgmulighederne er **Hjemmehørende** og **Ikke-hjemmehørende**.</span><span class="sxs-lookup"><span data-stu-id="9c852-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="9c852-113">I feltet **Sektionskode** skal du angive den sektionskode, der anvendes på TDS-komponentgruppen.</span><span class="sxs-lookup"><span data-stu-id="9c852-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="9c852-114">Luk siden.</span><span class="sxs-lookup"><span data-stu-id="9c852-114">Close the page.</span></span>
