[!include [banner](../../includes/banner.md)]

4. <span data-ttu-id="7aa3d-111">Skriv en værdi i feltet Beskrivelse.</span><span class="sxs-lookup"><span data-stu-id="7aa3d-111">In the Description field, type a value.</span></span>
