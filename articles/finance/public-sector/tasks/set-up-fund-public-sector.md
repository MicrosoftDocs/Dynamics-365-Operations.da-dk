# <a name="set-up-a-fund-in-the-public-sector"></a><span data-ttu-id="68b1f-103">Konfigurere en finansieringskilde i den offentlige sektor</span><span class="sxs-lookup"><span data-stu-id="68b1f-103">Set up a fund in the public sector</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="68b1f-104">Middeltyper skal være oprettet, før du kan konfigurere midler.</span><span class="sxs-lookup"><span data-stu-id="68b1f-104">Fund types must be created before you set up funds.</span></span> <span data-ttu-id="68b1f-105">Hvert enkelt middel skal have et navn og et entydigt nummer og skal være tildelt en middeltype og middelklasse.</span><span class="sxs-lookup"><span data-stu-id="68b1f-105">Each fund must have a name and a unique number, and must be assigned a fund type and fund class.</span></span> <span data-ttu-id="68b1f-106">Denne procedure er oprettet med PSUS-demodatafirmaet i den offentlige sektor partition.</span><span class="sxs-lookup"><span data-stu-id="68b1f-106">This procedure was created using the PSUS demo company data in the public sector partition.</span></span>

1. <span data-ttu-id="68b1f-107">Gå til Finans > Diagram over konti > Midler > Midler.</span><span class="sxs-lookup"><span data-stu-id="68b1f-107">Go to General ledger > Chart of accounts > Funds > Funds.</span></span>
2. <span data-ttu-id="68b1f-108">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="68b1f-108">Click New.</span></span>
3. <span data-ttu-id="68b1f-109">Indtast en værdi i feltet Middelnummer.</span><span class="sxs-lookup"><span data-stu-id="68b1f-109">In the Fund number field, type a value.</span></span>
4. <span data-ttu-id="68b1f-110">Skriv en værdi i feltet Middelnavn.</span><span class="sxs-lookup"><span data-stu-id="68b1f-110">In the Fund name field, type a value.</span></span>
5. <span data-ttu-id="68b1f-111">Klik på rullelisten i feltet Middeltype for at åbne opslaget.</span><span class="sxs-lookup"><span data-stu-id="68b1f-111">In the Fund type field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="68b1f-112">Vælg middeltypen for dette middel, på listen.</span><span class="sxs-lookup"><span data-stu-id="68b1f-112">In the list, select the fund type for this fund.</span></span>
7. <span data-ttu-id="68b1f-113">Vælg en indstilling i feltet Middelklasse.</span><span class="sxs-lookup"><span data-stu-id="68b1f-113">In the Fund class field, select an option.</span></span>
8. <span data-ttu-id="68b1f-114">Angiv indstillingen Finansieringskilde med indberetning til Ja eller Nej.</span><span class="sxs-lookup"><span data-stu-id="68b1f-114">Set the Non-reporting fund option to Yes or No.</span></span>
9. <span data-ttu-id="68b1f-115">Sæt indstillingen Overordnet finansieringskilde til Ja eller Nej.</span><span class="sxs-lookup"><span data-stu-id="68b1f-115">Set the Major fund option to Yes or No.</span></span>
10. <span data-ttu-id="68b1f-116">Sæt indstillingen Tilsidesæt valgte ultimoindstilling til Ja eller Nej.</span><span class="sxs-lookup"><span data-stu-id="68b1f-116">Set the Override selected year-end option to Yes or No.</span></span>
11. <span data-ttu-id="68b1f-117">Hvis du angiver indstillingen Tilsidesæt valgte ultimoindstilling til Ja, skal du vælge en ultimoindstilling.</span><span class="sxs-lookup"><span data-stu-id="68b1f-117">If you set the Override selected year-end option to Yes, select a year-end option.</span></span>
12. <span data-ttu-id="68b1f-118">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="68b1f-118">Click Save.</span></span>
