# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="e9262-103">Ressourceopfyldelse for planlagte ressourcer</span><span class="sxs-lookup"><span data-stu-id="e9262-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e9262-104">En projektleder kan planlægge nødvendig ressourceroller for et projekt.</span><span class="sxs-lookup"><span data-stu-id="e9262-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="e9262-105">Ressourcestyringen ser disse planlagte ressourcer som anmodninger på siden **Opfyldelse af ressource** og kan tildele faktiske ressourcer.</span><span class="sxs-lookup"><span data-stu-id="e9262-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="e9262-106">På siden **Alle projekter** skal du vælge projektet **XYZ-opgradering fase 2**.</span><span class="sxs-lookup"><span data-stu-id="e9262-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="e9262-107">Vælg **Projekt**, og vælg derefter **Rediger**.</span><span class="sxs-lookup"><span data-stu-id="e9262-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="e9262-108">Under fanen **Projektteam og planlægning** skal vælge **Tilføj**.</span><span class="sxs-lookup"><span data-stu-id="e9262-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="e9262-109">I dialogboksen **Tilføj roller** skal du vælg rollen **Softwareudvikler**.</span><span class="sxs-lookup"><span data-stu-id="e9262-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="e9262-110">Vælg **Opret**, og luk derefter projektsiden.</span><span class="sxs-lookup"><span data-stu-id="e9262-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="e9262-111">På siden **Opfyldelse af ressource** skal du vælge **Softwareudvikler 1** til projektet **XYZ-opgraderingsprojekt fase 2**.</span><span class="sxs-lookup"><span data-stu-id="e9262-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="e9262-112">Vælg en arbejder, og vælg derefter **Tildel**.</span><span class="sxs-lookup"><span data-stu-id="e9262-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="e9262-113">Kontroller, at linjen for **Softwareudvikler 1** er blevet fjernet for **XYZ-opgraderingsprojekt fase 2**.</span><span class="sxs-lookup"><span data-stu-id="e9262-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="e9262-114">Under fanen **Projektteam og planlægning** for projektet **XYZ-opgradering fase 2** skal du kontrollere, at den arbejder, som du valgte i det forrige trin, er blevet tilføjet som **Softwareudvikler**.</span><span class="sxs-lookup"><span data-stu-id="e9262-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
