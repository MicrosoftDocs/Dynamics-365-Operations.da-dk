# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="7a93e-103">Oprette en kreditnota på projektfakturaer</span><span class="sxs-lookup"><span data-stu-id="7a93e-103">Create a credit note on project invoices</span></span>

[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="7a93e-104">Gå til Projektstyring og regnskab > Projekter > Alle projekter.</span><span class="sxs-lookup"><span data-stu-id="7a93e-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="7a93e-105">Find og vælg den ønskede post på listen.</span><span class="sxs-lookup"><span data-stu-id="7a93e-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="7a93e-106">Vælg det projekt-id, der bruges til kreditnotaen.</span><span class="sxs-lookup"><span data-stu-id="7a93e-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="7a93e-107">Klik op linket i den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="7a93e-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="7a93e-108">Klik på Administrer i handlingsruden.</span><span class="sxs-lookup"><span data-stu-id="7a93e-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="7a93e-109">Klik på Fakturajournaler.</span><span class="sxs-lookup"><span data-stu-id="7a93e-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="7a93e-110">Vælg transaktionen til kreditnotaen.</span><span class="sxs-lookup"><span data-stu-id="7a93e-110">Select the transaction for credit note.</span></span> <span data-ttu-id="7a93e-111">Markér den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="7a93e-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="7a93e-112">Klik på Funktioner.</span><span class="sxs-lookup"><span data-stu-id="7a93e-112">Click Functions.</span></span> 
8. <span data-ttu-id="7a93e-113">Klik på Vælg til kreditnota.</span><span class="sxs-lookup"><span data-stu-id="7a93e-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="7a93e-114">Vælg transaktionen til kreditnotaen ved at markere afkrydsningsfeltet.</span><span class="sxs-lookup"><span data-stu-id="7a93e-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="7a93e-115">Klik på OK.</span><span class="sxs-lookup"><span data-stu-id="7a93e-115">Click OK.</span></span> 

<span data-ttu-id="7a93e-116">Efter dette trin er kreditnotaen tilgængelig for fakturaforslag.</span><span class="sxs-lookup"><span data-stu-id="7a93e-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
