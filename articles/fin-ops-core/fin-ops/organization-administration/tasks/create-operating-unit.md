# <a name="create-an-operating-unit"></a><span data-ttu-id="9edc1-103">Opret en driftsenhed</span><span class="sxs-lookup"><span data-stu-id="9edc1-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="9edc1-104">En driftsenhed er en organisation, der bruges til at opdele administrationen af økonomiske ressourcer og driftsprocesser i et firma.</span><span class="sxs-lookup"><span data-stu-id="9edc1-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="9edc1-105">Personer i en driftsenhed er forpligtet til at maksimere brugen af knappe ressourcer, til at forbedre processer og til at redegøre for deres præstation.</span><span class="sxs-lookup"><span data-stu-id="9edc1-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="9edc1-106">Typerne af driftsenheder inkluderer bærere, virksomhedsenheder, afdelinger og værdistrømme.</span><span class="sxs-lookup"><span data-stu-id="9edc1-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="9edc1-107">Du kan bruge følgende procedure for at oprette en driftsenhed.</span><span class="sxs-lookup"><span data-stu-id="9edc1-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="9edc1-108">Det demodatafirma, der bruges til at oprette denne procedure, er USMF.</span><span class="sxs-lookup"><span data-stu-id="9edc1-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="9edc1-109">Gå til **Navigationsrude > Moduler > Virksomhedsadministration > Organisationer > Driftsenheder**.</span><span class="sxs-lookup"><span data-stu-id="9edc1-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="9edc1-110">Klik på **Ny** for at åbne dialogboksen Fjern.</span><span class="sxs-lookup"><span data-stu-id="9edc1-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="9edc1-111">Find og vælg den ønskede post på listen.</span><span class="sxs-lookup"><span data-stu-id="9edc1-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="9edc1-112">Vælg den type driftsenhed, du vil oprette.</span><span class="sxs-lookup"><span data-stu-id="9edc1-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="9edc1-113">Klik op linket i den valgte række på listen.</span><span class="sxs-lookup"><span data-stu-id="9edc1-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="9edc1-114">Skriv en værdi i feltet **Navn**.</span><span class="sxs-lookup"><span data-stu-id="9edc1-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="9edc1-115">Udvid sektionen **Generelt**, hvis det er nødvendigt.</span><span class="sxs-lookup"><span data-stu-id="9edc1-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="9edc1-116">Angiv generelle oplysninger om driftsenheden, som f.eks. id-nummer, DUNS-nummer og leder.</span><span class="sxs-lookup"><span data-stu-id="9edc1-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="9edc1-117">Udvid sektionen **Adresser**, hvis det er nødvendigt.</span><span class="sxs-lookup"><span data-stu-id="9edc1-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="9edc1-118">Angiv adresseoplysninger, som f.eks. gadenavn og husnummer, postnummer og by.</span><span class="sxs-lookup"><span data-stu-id="9edc1-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="9edc1-119">Klik på **Tilføj** for at angive en ny adressepost, eller klik på Rediger for at ændre en eksisterende adressepost.</span><span class="sxs-lookup"><span data-stu-id="9edc1-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="9edc1-120">Udvid sektionen **Kontaktoplysninger**, hvis det er nødvendigt.</span><span class="sxs-lookup"><span data-stu-id="9edc1-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="9edc1-121">Angiv oplysninger om kommunikationsmåder, som f.eks. e-mail-adresser, URL-adresser og telefonnumre.</span><span class="sxs-lookup"><span data-stu-id="9edc1-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="9edc1-122">Klik på Ny for at angive en ny kommunikationspost.</span><span class="sxs-lookup"><span data-stu-id="9edc1-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="9edc1-123">Klik på **Flere indstillinger > Avanceret** for at ændre en eksisterende kommunikationspost.</span><span class="sxs-lookup"><span data-stu-id="9edc1-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="9edc1-124">Du kan evt. ændre **Driftsenhedsnummer**, hvis det er nødvendigt.</span><span class="sxs-lookup"><span data-stu-id="9edc1-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="9edc1-125">Bemærk, at dette tal er en entydig idenitifikator for den tilsvarende **Part**-post og må ikke være det samme som en anden driftsenhed.</span><span class="sxs-lookup"><span data-stu-id="9edc1-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="9edc1-126">Vælg **Gem**.</span><span class="sxs-lookup"><span data-stu-id="9edc1-126">Select **Save**.</span></span>
