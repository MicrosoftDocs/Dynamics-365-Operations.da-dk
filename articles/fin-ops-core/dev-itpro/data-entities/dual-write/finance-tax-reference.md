# <a name="access-to-finance-and-tax-reference-data"></a>Adgang til finansierings- og momsreferencedata

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

Hver virksomhed arbejder med et grundlæggende sæt økonomiske data, f.eks. regnskabskalenderåret, den valuta, som virksomheden handler i, og de konti, der indeholder de penge, som kommer ind i og ud af virksomheden, momssatser og remittering. Disse data findes i Finance and Operations-apps. De eksponeres imidlertid i Common Data Service, så modeldrevne apps i Microsoft Dynamics 365 kan have en enkelt kilde til finansierings- og momsdata. På denne måde er data ensartede i hele virksomhedens økosystem. 

Finans- og momsdata integreres ved hjælp af følgende tilknytninger:

+ [Integreret finans](ledger-mapping.md)
+ [Integreret momsmaster](tax-mapping.md)
