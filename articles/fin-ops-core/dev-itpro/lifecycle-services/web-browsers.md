# <a name="supported-web-browsers-for-lifecycle-services"></a>Understøttede webbrowsere til Lifecycle Services 

[!include[banner](../includes/banner.md)]

Brugere kan få adgang til Microsoft Dynamics Lifecyle Services (LCS) ved hjælp af de nyeste versioner af disse populære browsere: 

- Microsoft Edge (anbefales: [Chromium-baseret Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Gå til producentens websted for at finde den nyeste version af hver webbrowser. For at få optimal ydeevne og optimal oplevelse anbefales det, at du bruger den seneste version af en nyere browser, især Microsoft Edge.
