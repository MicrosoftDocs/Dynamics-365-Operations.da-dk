# <a name="list-of-er-functions-in-the-record-category"></a>Liste over ER-funktioner i kategorien post

[!include [banner](../includes/banner.md)]

Postfunktioner til elektronisk rapportering (ER) kan bruges til at udtrække oplysninger fra og udføre handlinger på datakilder for datatypen *Container (post)*. Denne artikel indeholder en oversigt over disse funktioner.

## <a name="list-of-supported-functions"></a>Liste med understøttede funktioner

| Funktion | Beskrivelse |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Denne funktion returnerer en nulværdi for *Container (post)*, som har samme struktur, som den angivne postliste eller post. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Denne funktion returnerer en nulværdi for *Container (post)*, som har samme struktur, som den angivne postliste eller post. |
