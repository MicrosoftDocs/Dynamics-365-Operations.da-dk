# <a name="upload-files-other-than-images-and-videos"></a>Upload andre filer end billeder og videoer

[!include [banner](includes/banner.md)]

I dette emne beskrives, hvordan du kan uploade andre filer end billeder og videoer i Microsoft Dynamics 365 Commerce-webstedsgenerator.

## <a name="overview"></a>Overblik

Mediebiblioteket i Commerce-webstedsgenerator understøtter upload af andre binære aktiver end billeder eller videoer. Du kan f.eks. uploade Microsoft Excel-, Microsoft Word-, Microsoft PowerPoint- eller PDF-filer.

Følgende dokumenttyper understøttes:
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>Overfør en fil

Følg disse trin for at uploade en fil til Commerce-webstedsgenerator.

1. Vælg **Mediebibliotek** i navigationsruden til venstre.
1. Vælg **Upload \> Upload medieelementer** på kommandolinjen.
1. Vælg en eller flere filer i Stifinder, og vælg derefter **Åbn**.
1. Angiv titel, beskrivelse og nøgleordsmetadata i dialogboksen **Upload medieelement** efter behov.
1. Hvis du vil udgive en eller flere filer umiddelbart efter upload, skal du markere afkrydsningsfeltet **Publicer medieelementer efter upload** .
1. Vælg **OK**.
