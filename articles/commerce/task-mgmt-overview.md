<span data-ttu-id="40138-104">Dette emne indeholder en oversigt over opgavestyringen for ledere og arbejdere i Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="40138-104">This topic provides an overview of task management for managers and workers in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="40138-105">Oversigt</span><span class="sxs-lookup"><span data-stu-id="40138-105">Overview</span></span>

<span data-ttu-id="40138-106">I et detailmiljø er det altid vanskeligt at sikre, at opgaver udføres den rigtige person på det rigtige tidspunkt.</span><span class="sxs-lookup"><span data-stu-id="40138-106">In a retail environment, it's always difficult to make sure that tasks are performed by the right person at the right time.</span></span> <span data-ttu-id="40138-107">Detailhandlere skal kunne give arbejdere besked om kommende opgaver og oprette relaterede forretningskontekster, så opgaverne kan udføres korrekt og til tiden.</span><span class="sxs-lookup"><span data-stu-id="40138-107">Retailers must be able to notify workers about upcoming tasks and provide related business context, so that the tasks can be completed correctly and on time.</span></span>

<span data-ttu-id="40138-108">Opgavestyring er en produktivitetsfunktion i Dynamics 365 Commerce, der gør det muligt for ledere og medarbejdere at oprette opgavelister, administrere tildelingskriterier, registrere opgavens status og integrere disse handlinger mellem Commerce-bagkontoret og POS-programmer.</span><span class="sxs-lookup"><span data-stu-id="40138-108">Task management is a productivity feature in Dynamics 365 Commerce that lets managers and workers create task lists, manage assignment criteria, track task status, and integrate these operations between Commerce back office and point of sale (POS) applications.</span></span>

<span data-ttu-id="40138-109">Headquarters-personer kan bruge opgavestyring til at oprette opgavelister til detailbutikker og til at spore status pr. butik eller arbejder.</span><span class="sxs-lookup"><span data-stu-id="40138-109">Headquarters personas can use task management to create task lists for retail stores, and to track status by store or worker.</span></span> <span data-ttu-id="40138-110">De kan også oprette tilbagevendende opgaver (f.eks. "Tjekliste til lukning torsdag aften").</span><span class="sxs-lookup"><span data-stu-id="40138-110">They can also create recurrent tasks (for example, "Thursday night closing checklist").</span></span>

<span data-ttu-id="40138-111">Butikschefer kan bruge opgavestyring til at tildele opgaver til individuelle arbejdere, sende beskeder om kommende opgaver eller opgaver, der er forfaldne, opdatere opgavestatus og oprette opgaver med et enkelt formål i POS-programmet.</span><span class="sxs-lookup"><span data-stu-id="40138-111">Store managers can use task management to assign tasks to individual workers, send notifications about upcoming tasks or tasks that are past due, update task status, and create single-purpose tasks in the POS application.</span></span> <span data-ttu-id="40138-112">Arbejderne kan derefter se beskeder, få vist opgavedetaljer og opdatere opgavestatus på kasseapparatet.</span><span class="sxs-lookup"><span data-stu-id="40138-112">Workers can then see notifications, view task details, and update task status at the POS.</span></span>
