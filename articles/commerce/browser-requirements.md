# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Krav til browser for Dynamics 365 Commerce-websteder

[!include [banner](includes/banner.md)]

Denne artikel viser browserkravene til Microsoft Dynamics 365 Commerce-websteder.

Dynamics 365 Commerce-websteder understøttes bedst af følgende webbrowsere. Gå til producentens websted for at finde den nyeste version af hver webbrowser.

- [Microsoft Edge](https://www.microsoft.com/edge) (op til den aktuelle version minus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (op til den aktuelle version minus 1)
- [Google Chrome](https://www.google.com/chrome) (op til den aktuelle version minus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (op til den aktuelle version minus 1)

> [!NOTE]
> Internet Explorer understøttes ikke. Hvis du bruger Internet Explorer, vil nogle funktioner muligvis ikke fungere, og brugerne vil muligvis have en understandardoplevelse.
