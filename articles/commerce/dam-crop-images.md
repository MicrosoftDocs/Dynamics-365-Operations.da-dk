## <a name="overview"></a><span data-ttu-id="831a6-105">Overblik</span><span class="sxs-lookup"><span data-stu-id="831a6-105">Overview</span></span>

<span data-ttu-id="831a6-106">Med mediebiblioteket i Commerce-webstedsgenerator kan du beskære billeder for at optimere dem til forskellige modultyper og billeder.</span><span class="sxs-lookup"><span data-stu-id="831a6-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="831a6-107">Beskære et billede</span><span class="sxs-lookup"><span data-stu-id="831a6-107">Crop an image</span></span>

<span data-ttu-id="831a6-108">Følg disse trin for at beskære et billede i webstedsgenerator.</span><span class="sxs-lookup"><span data-stu-id="831a6-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="831a6-109">Vælg **Mediebibliotek** i venstre navigationsrude i Commerce-webstedgenerator.</span><span class="sxs-lookup"><span data-stu-id="831a6-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="831a6-110">Vælg det billede, du vil redigere, i hovedvinduet.</span><span class="sxs-lookup"><span data-stu-id="831a6-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="831a6-111">Vælg **Rediger** på kommandolinjen.</span><span class="sxs-lookup"><span data-stu-id="831a6-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="831a6-112">Vælg billedet for at skifte til **Redigeringstilstand**.</span><span class="sxs-lookup"><span data-stu-id="831a6-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="831a6-113">Vælg **Rediger visning efter modul** i **Redigeringstilstand**.</span><span class="sxs-lookup"><span data-stu-id="831a6-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="831a6-114">Vælg modultypen i rullemenuen **Modul**.</span><span class="sxs-lookup"><span data-stu-id="831a6-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="831a6-115">Vælg visningstypen i rullemenuen **Visningstype**.</span><span class="sxs-lookup"><span data-stu-id="831a6-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="831a6-116">Vælg billedplaceringen i rullelisten **Placering**.</span><span class="sxs-lookup"><span data-stu-id="831a6-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="831a6-117">Vælg billedstørrelsen i rullemenuen **Billede**.</span><span class="sxs-lookup"><span data-stu-id="831a6-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="831a6-118">Billedet overlejres med det område, der repræsenterer beskæringsområdet.</span><span class="sxs-lookup"><span data-stu-id="831a6-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="831a6-119">Flyt og tilpas beskæringsområdet efter behov.</span><span class="sxs-lookup"><span data-stu-id="831a6-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="831a6-120">Højde-bredde-forholdet vil automatisk blive bevaret.</span><span class="sxs-lookup"><span data-stu-id="831a6-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="831a6-121">Når du er færdig, skal du vælge **Gem** på kommandolinjen og derefter vælge **Afslut redigering**.</span><span class="sxs-lookup"><span data-stu-id="831a6-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="831a6-122">Når den brugerdefinerede beskæring er fuldført, træder billedændringerne i kraft næsten med det samme.</span><span class="sxs-lookup"><span data-stu-id="831a6-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
