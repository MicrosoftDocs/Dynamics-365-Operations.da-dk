# <a name="configure-the-functionality-profile-for-a-sales-representative"></a><span data-ttu-id="a7b15-103">Konfigurere funktionalitetsprofilen for en sælger</span><span class="sxs-lookup"><span data-stu-id="a7b15-103">Configure the functionality profile for a sales representative</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a7b15-104">Denne fremgangsmåde viser, hvordan du konfigurerer de indstillinger for en butiks funktionalitetsprofil, der gælder for sælgere, som bruger USRT-demodatafirmaet.</span><span class="sxs-lookup"><span data-stu-id="a7b15-104">This procedure demonstrates how to configure a store's functionality profile settings that apply to sales representatives using the USRT demo data company.</span></span>

1. <span data-ttu-id="a7b15-105">Gå til Retail og Commerce > Konfiguration af kanal > POS-opsætning > POS-profiler > Funktionalitetsprofiler.</span><span class="sxs-lookup"><span data-stu-id="a7b15-105">Go to Retail and Commerce > Channel setup > POS setup > POS profiles > Functionality profiles.</span></span>
2. <span data-ttu-id="a7b15-106">Klik på Rediger.</span><span class="sxs-lookup"><span data-stu-id="a7b15-106">Click Edit.</span></span>
3. <span data-ttu-id="a7b15-107">Udvid afsnittet Funktioner.</span><span class="sxs-lookup"><span data-stu-id="a7b15-107">Expand the Functions section.</span></span>
    * <span data-ttu-id="a7b15-108">Du kan bruge indstillingerne for funktionalitetsprofilen til at konfigurere POS til automatisk at tilføje kassererens standardsalgsgruppe, til at bede om salgsgrupper og til at kræve salgsgrupper.</span><span class="sxs-lookup"><span data-stu-id="a7b15-108">You can use the functionality profile settings to configure POS to automatically add the cashier's default sales group, to prompt for sales groups, and to require sales groups.</span></span>  
4. <span data-ttu-id="a7b15-109">Vælg en indstilling i feltet Spørg efter sælger.</span><span class="sxs-lookup"><span data-stu-id="a7b15-109">In the Prompt for sales representative field, select an option.</span></span>
5. <span data-ttu-id="a7b15-110">Vælg Ja i feltet Kræv sælger.</span><span class="sxs-lookup"><span data-stu-id="a7b15-110">Select Yes in the Require sales representative field.</span></span>
