## <a name="overview"></a><span data-ttu-id="3b22a-105">Oversigt</span><span class="sxs-lookup"><span data-stu-id="3b22a-105">Overview</span></span>

<span data-ttu-id="3b22a-106">Når du opretter en Excel-fil, så du kan redigere detailtransaktioner, udfyldes filen med nogle standardfelter.</span><span class="sxs-lookup"><span data-stu-id="3b22a-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="3b22a-107">Hvis et felt, der skal opdateres, som standard ikke vises i den oprettede Excel-fil, kan du tilføje det.</span><span class="sxs-lookup"><span data-stu-id="3b22a-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="3b22a-108">Føje felter til et regneark i en Excel-projektmappe</span><span class="sxs-lookup"><span data-stu-id="3b22a-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="3b22a-109">Hvis du vil føje felter til en Excel-projektmappe, så du kan redigere detailtransaktioner, skal du følge disse trin.</span><span class="sxs-lookup"><span data-stu-id="3b22a-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="3b22a-110">Overfør og åbn Excel-filen fra siden **Opgørelser**, siden **Detailtransaktioner** eller feltet **Transaktionsvalideringsfejl** i arbejdsområdet **Butiksøkonomi**.</span><span class="sxs-lookup"><span data-stu-id="3b22a-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="3b22a-111">Vælg **Design**.</span><span class="sxs-lookup"><span data-stu-id="3b22a-111">Select **Design**.</span></span>
1. <span data-ttu-id="3b22a-112">Vælg blyantsymbolet for den ønskede tabel, og find og markér derefter det felt, du vil tilføje, på listen over tilgængelige felter.</span><span class="sxs-lookup"><span data-stu-id="3b22a-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
1. <span data-ttu-id="3b22a-113">Vælg **Tilføj**, og vælg derefter **Opdater**.</span><span class="sxs-lookup"><span data-stu-id="3b22a-113">Select **Add**, and then select **Update**.</span></span> <span data-ttu-id="3b22a-114">Du kan ændre rækkefølgen for felterne.</span><span class="sxs-lookup"><span data-stu-id="3b22a-114">You can reorder fields.</span></span>
1. <span data-ttu-id="3b22a-115">Når opdateringen er fuldført, skal du vælge **Opdater** for at hente dataene til den nye kolonne.</span><span class="sxs-lookup"><span data-stu-id="3b22a-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="3b22a-116">Det nye felt og de tilhørende data burde nu kunne redigeres i Excel.</span><span class="sxs-lookup"><span data-stu-id="3b22a-116">The new field and data for it should now be available for editing in Excel.</span></span>
