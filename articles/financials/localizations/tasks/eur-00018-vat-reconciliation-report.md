[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="55053-104">Denne fremgangsmåde viser, hvordan du kan oprette rapporten over momsafstemning, der er tilgængelig for islandske og norske juridiske enheder.</span><span class="sxs-lookup"><span data-stu-id="55053-104">This procedure shows how to generate the tax reconciliation report that is available for Icelandic and Norwegian legal entities.</span></span> <span data-ttu-id="55053-105">Denne opgave blev oprettet ved hjælp af demodatafirmaet DEMF med landet/området i den juridiske enheds primære adresse opdateret til Norge.</span><span class="sxs-lookup"><span data-stu-id="55053-105">This task was created using the demo data company DEMF with the country/region of legal entity primary address updated to be Norway.</span></span>

2. <span data-ttu-id="55053-107">Indtast eller vælg en værdi i feltet Fra konto.</span><span class="sxs-lookup"><span data-stu-id="55053-107">In the From account field, enter or select a value.</span></span>
    * <span data-ttu-id="55053-108">Vælg værdien 200140.</span><span class="sxs-lookup"><span data-stu-id="55053-108">Select the value 200140.</span></span>  
3. <span data-ttu-id="55053-109">Indtast eller vælg en værdi i feltet Til konto.</span><span class="sxs-lookup"><span data-stu-id="55053-109">In the To account field, enter or select a value.</span></span>
    * <span data-ttu-id="55053-110">Vælg værdien 600180.</span><span class="sxs-lookup"><span data-stu-id="55053-110">Select the value 600180.</span></span>  
4. <span data-ttu-id="55053-111">Indtast eller vælg en værdi i feltet Afregningsperiode.</span><span class="sxs-lookup"><span data-stu-id="55053-111">In the Settlement period field, enter or select a value.</span></span>
    * <span data-ttu-id="55053-112">Vælg værdien Man.</span><span class="sxs-lookup"><span data-stu-id="55053-112">Select the value Mon.</span></span>  
5. <span data-ttu-id="55053-113">Indtast en dato i feltet Fra dato.</span><span class="sxs-lookup"><span data-stu-id="55053-113">In the From date field, enter a date.</span></span>
    * <span data-ttu-id="55053-114">Vælg værdien 01-11-2015.</span><span class="sxs-lookup"><span data-stu-id="55053-114">Select the value 2015-11-01.</span></span>  
6. <span data-ttu-id="55053-115">Vælg Ja i feltet Vis detaljer.</span><span class="sxs-lookup"><span data-stu-id="55053-115">Select Yes in the Show details field.</span></span>
7. <span data-ttu-id="55053-116">Klik på OK.</span><span class="sxs-lookup"><span data-stu-id="55053-116">Click OK.</span></span>
