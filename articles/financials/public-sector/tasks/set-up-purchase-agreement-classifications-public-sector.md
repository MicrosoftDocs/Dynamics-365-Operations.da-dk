# <a name="set-up-purchase-agreement-classifications-in-the-public-sector"></a><span data-ttu-id="5002d-103">Konfigurere klassifikationer for købsaftaler i den offentlige sektor</span><span class="sxs-lookup"><span data-stu-id="5002d-103">Set up purchase agreement classifications in the public sector</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="5002d-104">Købsaftaleklassifikationen gør det muligt at styre de administrative oplysninger, der findes i købsaftaler.</span><span class="sxs-lookup"><span data-stu-id="5002d-104">The purchase agreement classification allows you to control the administrative information that is available on purchase agreements.</span></span> 

<span data-ttu-id="5002d-105">Denne procedure er oprettet for den franske offentlige sektor med data fra PSUS-demofirmaet i den offentlige sektor partition.</span><span class="sxs-lookup"><span data-stu-id="5002d-105">This procedure was created for the French public sector using the PSUS demo company data in the public sector partition.</span></span>

1. <span data-ttu-id="5002d-106">Gå til Indkøb og forsyning > Opsætning > Købsaftaleklassifikation.</span><span class="sxs-lookup"><span data-stu-id="5002d-106">Go to Procurement and sourcing > Setup > Purchase agreement classification.</span></span>
2. <span data-ttu-id="5002d-107">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="5002d-107">Click New.</span></span>
3. <span data-ttu-id="5002d-108">Skriv en værdi i feltet Navn.</span><span class="sxs-lookup"><span data-stu-id="5002d-108">In the Name field, type a value.</span></span>
4. <span data-ttu-id="5002d-109">Skriv en værdi i feltet Beskrivelse.</span><span class="sxs-lookup"><span data-stu-id="5002d-109">In the Description field, type a value.</span></span>
5. <span data-ttu-id="5002d-110">Valgfrit: Vælg indstillingen Underleverandører for at medtage oplysninger om underleverandører.</span><span class="sxs-lookup"><span data-stu-id="5002d-110">Optional: Select the Subcontractors option to include information about subcontractors.</span></span>
    * <span data-ttu-id="5002d-111">En købsaftale kan have en primære kontrahent, medkontrahenter og underleverandører.</span><span class="sxs-lookup"><span data-stu-id="5002d-111">A purchase agreement can have a prime contractor, co-contractors, and subcontractors.</span></span> <span data-ttu-id="5002d-112">Du kan tildele den primære kontrahent til den overordnede købsaftale og en medkontrahent eller underleverandør til specifikke underordnede aftaler.</span><span class="sxs-lookup"><span data-stu-id="5002d-112">You can assign the prime contractor to the parent purchase agreement and a co-contractor or subcontractor to specific child agreements.</span></span>  
6. <span data-ttu-id="5002d-113">Valgfrit: Vælg indstillingen Certificeringer for at medtage oplysninger om certificeringer for kreditorer.</span><span class="sxs-lookup"><span data-stu-id="5002d-113">Optional: Select the Certifications option to include information about certifications for vendors.</span></span>
    * <span data-ttu-id="5002d-114">Certificeringsoplysninger kan bruges til at oprette en rapport, som du kan bruge til at overvåge kreditorens overholdelse af bestemte krav.</span><span class="sxs-lookup"><span data-stu-id="5002d-114">Certification information can be used to generate a report that lets you monitor vendor compliance with certification requirements.</span></span>  
7. <span data-ttu-id="5002d-115">Valgfrit: Vælg indstillingen Kræv direkte fakturering for at forhindre brug af aftræksordrer.</span><span class="sxs-lookup"><span data-stu-id="5002d-115">Optional: Select the Require direct invoicing option to prevent the use of release orders.</span></span>
8. <span data-ttu-id="5002d-116">Valgfrit: Vælg indstillingen Aktiviteter for at medtage oplysninger om opgaver og milepæle.</span><span class="sxs-lookup"><span data-stu-id="5002d-116">Optional: Select the Activities option to include information about tasks and milestones.</span></span>
9. <span data-ttu-id="5002d-117">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="5002d-117">Click Save.</span></span>
