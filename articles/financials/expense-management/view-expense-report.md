# <a name="view-an-expense-report"></a><span data-ttu-id="117a1-103">Vis en udgiftsrapport</span><span class="sxs-lookup"><span data-stu-id="117a1-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="117a1-104">Når du kører en forespørgsel for at få vist posteringer i Finans eller Indkøb og forsyning, returnerer forespørgslen muligvis for mange oplysninger.</span><span class="sxs-lookup"><span data-stu-id="117a1-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="117a1-105">Derfor kan du evt. indsnævre søgningen.</span><span class="sxs-lookup"><span data-stu-id="117a1-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="117a1-106">For at finde de oplysninger, du behøver, kan du være nødt til at foretage en detailudledning af en postering for at se den udgiftsrapport, som posteringen forekom på.</span><span class="sxs-lookup"><span data-stu-id="117a1-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="117a1-107">Hvad angår udgiftsrapporter, kan du foretage en detailudledning fra en postering til den oprindelige rapport, der gemmes i Udgiftsstyring.</span><span class="sxs-lookup"><span data-stu-id="117a1-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="117a1-108">Ud over det samlede udgiftsbeløb kan du få vist specificerede posteringer, posteringer, der er berettiget til momsrefusion, og de kvitteringer, der er tilknyttet udgifterne.</span><span class="sxs-lookup"><span data-stu-id="117a1-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="117a1-109">Hvis du vil foretage detailudledning i den oprindelige udgiftsrapport, som en transaktion blev vist i, skal du vælge transaktionen og derefter vælge **Originaldokument**.</span><span class="sxs-lookup"><span data-stu-id="117a1-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="117a1-110">Når du får vist enkelte posteringer, kan du også se eventuelt andre data, der er tilknyttet en postering.</span><span class="sxs-lookup"><span data-stu-id="117a1-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="117a1-111">For en udspecificeret hoteludgift kan du f.eks. se en detaljeret liste over udgifter, som f.eks. internetadgang, telefonopkald og room service.</span><span class="sxs-lookup"><span data-stu-id="117a1-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
