# <a name="fixed-asset-mass-update"></a><span data-ttu-id="d0351-103">Masseopdatere anlægsaktiver</span><span class="sxs-lookup"><span data-stu-id="d0351-103">Fixed asset mass update</span></span>

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="d0351-104">Hvis du bruger modeller, kan du ændre afskrivningsprincipperne for grupper af aktiver, der indgår i samme model.</span><span class="sxs-lookup"><span data-stu-id="d0351-104">If you use books, you can change the depreciation conventions for groups of assets that are part of the same book.</span></span>

<span data-ttu-id="d0351-105">Hvis du f.eks. befinder dig i USA, og du placerer mere end 40 procent dine aktiver inden for service i fjerde kvartal af året, skal du bruge afskrivningsprincippet Midt i kvartal.</span><span class="sxs-lookup"><span data-stu-id="d0351-105">For example, if you are in the United States, and you put more than 40 percent of your assets in service during the fourth quarter of the year, you must use the mid-quarter depreciation convention.</span></span> <span data-ttu-id="d0351-106">Du kan bruge funktionen til masseopdatering til at ændre alle aktiver, der kræver det nye afskrivningsprincip.</span><span class="sxs-lookup"><span data-stu-id="d0351-106">You can use the process for a mass update to change all assets that require the new depreciation convention.</span></span> 

<span data-ttu-id="d0351-107">Når du opdaterer afskrivningsprincippet for aktiverne, sletter du alle eksisterende afskrivningsposteringer for disse aktiver.</span><span class="sxs-lookup"><span data-stu-id="d0351-107">When you update the depreciation convention for assets, you delete all depreciation transactions that exist for those assets.</span></span> <span data-ttu-id="d0351-108">Du kan også slette alle posteringer for afkrivningsreguleringer, posteringer til straksafskrivning og posteringer for ekstraordinær afskrivning for disse aktiver.</span><span class="sxs-lookup"><span data-stu-id="d0351-108">You also delete all transactions for depreciation adjustments, transactions for bonus depreciation, and transactions for extraordinary depreciation for those assets.</span></span> 

<span data-ttu-id="d0351-109">Hvis du vil opdatere afskrivningsprincippet for aktiver, der allerede er afhændet, skal du først slette eksisterende kassationsposteringer.</span><span class="sxs-lookup"><span data-stu-id="d0351-109">To update the depreciation convention for assets that have already been disposed of, you must first delete the existing disposal transactions.</span></span> <span data-ttu-id="d0351-110">Du skal også slette alle de posteringer, der blev oprettet på grund af kassationsprocessen.</span><span class="sxs-lookup"><span data-stu-id="d0351-110">You must also delete all transactions that were generated because of the disposal process.</span></span> 

<span data-ttu-id="d0351-111">Når du har opdateret afskrivningsprincippet for aktiverne, kan du behandle afskrivning og ekstraordinær afskrivning for hvert enkelt anlægsaktiv.</span><span class="sxs-lookup"><span data-stu-id="d0351-111">After you update the depreciation convention for assets, you can process depreciation and extraordinary depreciation for each asset.</span></span> <span data-ttu-id="d0351-112">Du kan også udføre manuelle afskrivningsreguleringer, hvis det er nødvendigt.</span><span class="sxs-lookup"><span data-stu-id="d0351-112">You can also make manual depreciation adjustments, if any adjustments are required.</span></span>
