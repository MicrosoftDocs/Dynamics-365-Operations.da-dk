# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="5e492-103">Fjerne et anlægsaktiv ved hjælp af en fritekstfaktura</span><span class="sxs-lookup"><span data-stu-id="5e492-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="5e492-104">Denne fremgangsmåde viser, hvordan du kasserer et anlægsaktiv ved hjælp af fritekstfakturaen.</span><span class="sxs-lookup"><span data-stu-id="5e492-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="5e492-105">Gå til Debitor > Fakturaer > Alle fritekstfakturaer.</span><span class="sxs-lookup"><span data-stu-id="5e492-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="5e492-106">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="5e492-106">Click New.</span></span>
3. <span data-ttu-id="5e492-107">Indtast eller vælg en værdi i feltet Kundekonto.</span><span class="sxs-lookup"><span data-stu-id="5e492-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="5e492-108">Kontrollér standardfakturadatoen, og rediger den evt.</span><span class="sxs-lookup"><span data-stu-id="5e492-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="5e492-109">Kontrollér resterende standardoverskriftsfelter som f.eks. Valuta, og rediger evt.</span><span class="sxs-lookup"><span data-stu-id="5e492-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="5e492-110">Indtast en beskrivelse på fakturalinjen.</span><span class="sxs-lookup"><span data-stu-id="5e492-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="5e492-111">Indtast eller vælg hovedkontoen til fakturalinjen.</span><span class="sxs-lookup"><span data-stu-id="5e492-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="5e492-112">Kontrollér standardmomsgruppen og varemomsgruppen.</span><span class="sxs-lookup"><span data-stu-id="5e492-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="5e492-113">Angiv enhedsprisen eller beløbet for salg af anlægsaktivet.</span><span class="sxs-lookup"><span data-stu-id="5e492-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="5e492-114">Klik på Linjedetaljer.</span><span class="sxs-lookup"><span data-stu-id="5e492-114">Click Line details.</span></span>  
11. <span data-ttu-id="5e492-115">Vælg anlægsaktivnummeret, der skal sælges.</span><span class="sxs-lookup"><span data-stu-id="5e492-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="5e492-116">Klik på Bogfør.</span><span class="sxs-lookup"><span data-stu-id="5e492-116">Click Post.</span></span>
