<span data-ttu-id="7a847-104">Denne opgaveguide fører dig gennem forskellige metoder, der bruges til at oprette kreditorbetalinger, herunder hvordan du bruger et betalingsforslag eller manuelt angiver en engangsbetaling.</span><span class="sxs-lookup"><span data-stu-id="7a847-104">This task guide will walk you through various methods used to create vendor payments, including how to use a payment proposal or manually entering a one-off payment.</span></span> <span data-ttu-id="7a847-105">Denne procedure bruger demofirmaet USMF.</span><span class="sxs-lookup"><span data-stu-id="7a847-105">This procedure uses the USMF demo company.</span></span>

14. <span data-ttu-id="7a847-127">Klik på OK.</span><span class="sxs-lookup"><span data-stu-id="7a847-127">Click OK.</span></span>
    * <span data-ttu-id="7a847-128">Når du klikker på OK, vises resultaterne af forespørgslen.</span><span class="sxs-lookup"><span data-stu-id="7a847-128">After clicking OK, the results of the query will appear.</span></span> <span data-ttu-id="7a847-129">Hvis du ikke vil have vist listen over fakturaer, der er valgt til betaling, kan du gå tilbage til oversigtspanelet Parametre og ændre indstillingen Opret betalinger uden projektfaktura = Ja.</span><span class="sxs-lookup"><span data-stu-id="7a847-129">If you don't want to preview the list of invoices selected to pay, you can go back to the Parameters fast tab and change the setting Create payments without invoice preview = Yes.</span></span>
