# <a name="set-up-an-appointment-for-a-load"></a>Konfigurere en aftale for en last

[!include [banner](../../includes/banner.md)]

Denne procedure viser, hvordan du opretter og planlægger området for læsning. Denne konfiguration vil normalt blive udført af en transportkoordinator. Du kan bruge denne procedure i USMF-demodatafirmaet.


## <a name="select-the-load"></a>Vælg lasten
1. Gå til Transportstyring > Planlægning > Lastplanlægningspanel.
2. Fjern markeringen i afkrydsningsfeltet Skjul leverede og modtagne.
3. På listen skal du vælge den last, der har statussen Afsendt.
4. Klik på Transport.
5. Klik på Planlægning af aftale.

## <a name="create-an-appointment"></a>Opret en aftale
1. Klik på Ny.
2. Indtast eller vælg en værdi i feltet Aftaleregel.
3. Klik på Gem.
4. Klik på Opdater status.
5. Klik på Autoriser.
6. Klik på Gem.
7. Luk siden.
