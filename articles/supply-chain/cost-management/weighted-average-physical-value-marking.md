# <a name="weighted-average-with-physical-value-and-marking"></a><span data-ttu-id="eb80e-103">Vægtet gennemsnit med fysisk værdi og afmærkning</span><span class="sxs-lookup"><span data-stu-id="eb80e-103">Weighted average with physical value and marking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="eb80e-104">Vægtet gennemsnit er en lagermodel, der er baseret på princippet for vægtet gennemsnit, hvor afgange fra lager værdisættes til gennemsnitsværdien for varer, der er modtaget på lager i lagerlukningsperioden, samt en eventuel disponibel lagerbeholdning fra forrige periode.</span><span class="sxs-lookup"><span data-stu-id="eb80e-104">Weighted average is an inventory model based on the weighted average principle, where issues from inventory are valued at the average value of the items that are received into inventory during the inventory closing period, plus any on-hand inventory from the previous period.</span></span>

<span data-ttu-id="eb80e-105">Når du kører en lagerlukning, udlignes alle tilgange mod en virtuel afgang, der indeholder den samlede modtagne mængde og værdi.</span><span class="sxs-lookup"><span data-stu-id="eb80e-105">When you run an inventory closing, all receipts are settled against a virtual issue, which holds the total received quantity and value.</span></span> <span data-ttu-id="eb80e-106">Denne virtuelle afgang har en tilsvarende virtuel tilgang, som tilgangene vil blive udlignet fra.</span><span class="sxs-lookup"><span data-stu-id="eb80e-106">This virtual issue has a corresponding virtual receipt from which the issues are settled.</span></span> <span data-ttu-id="eb80e-107">På denne måde opnår alle afgange samme gennemsnitskostpris.</span><span class="sxs-lookup"><span data-stu-id="eb80e-107">In this manner, all issues get the same average cost.</span></span> <span data-ttu-id="eb80e-108">Den virtuelle afgang og tilgang kan ses som en virtuel overførsel, som kaldes "lagerlukningsoverførsel efter vægtet gennemsnit".</span><span class="sxs-lookup"><span data-stu-id="eb80e-108">The virtual issue and receipt can be seen as a virtual transfer, which is named the weighted average inventory closing transfer.</span></span>

<span data-ttu-id="eb80e-110">Når du bruger vægtet gennemsnit, kan du vælge at markere lagertransaktioner, så en specifik varetilgang udlignes mod en specifik afgang, i stedet for at bruge reglen om vægtet gennemsnit.</span><span class="sxs-lookup"><span data-stu-id="eb80e-110">When using weighted average, you can mark inventory transactions so that a specific item receipt is settled against a specific issue, instead of using the weighted average rule.</span></span>
