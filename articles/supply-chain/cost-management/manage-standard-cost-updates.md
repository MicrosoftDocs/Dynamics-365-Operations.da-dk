<span data-ttu-id="8d4a7-104">Opdateringer til standardomkostningsoplysninger kan administreres ved hjælp af to forskellige metoder – metoden med én version eller metoden med to versioner.</span><span class="sxs-lookup"><span data-stu-id="8d4a7-104">Updates to standard cost data can be managed by using two different approaches - the one-version approach or the two-version approach.</span></span> 

<span data-ttu-id="8d4a7-105">Med metoden med én version bruges en enkelt efterkalkulationsversion, der indeholder alle omkostningsposter.</span><span class="sxs-lookup"><span data-stu-id="8d4a7-105">The one-version approach uses a single costing version that contains all cost records.</span></span> <span data-ttu-id="8d4a7-106">Disse poster indeholder de oprindelige omkostninger og alle omkostningsopdateringer.</span><span class="sxs-lookup"><span data-stu-id="8d4a7-106">These records include the original costs and all cost updates.</span></span>

<span data-ttu-id="8d4a7-107">Med metoden med to versioner bruges én version, der indeholder posterne med de oprindelige omkostninger, og en anden version, der indeholder poster med alle omkostningsopdateringer.</span><span class="sxs-lookup"><span data-stu-id="8d4a7-107">The two-version approach uses one version that contains records of the original costs and a second version that contains records of all cost updates.</span></span> <span data-ttu-id="8d4a7-108">En væsentlig fordel ved metoden med to versioner er den klare fremstilling og sporing af omkostningsopdateringer i en særskilt efterkalkulationsversion, uden at den oprindelige efterkalkulationsversion påvirkes.</span><span class="sxs-lookup"><span data-stu-id="8d4a7-108">A primary advantage of the two-version approach is the clear delineation and tracking of cost updates in a separate costing version, without affecting the original costing version.</span></span> <span data-ttu-id="8d4a7-109">Metoden med to versioner kan bruges til at identificere flere trinvise opdateringer, hvor de enkelte trinvise opdateringer har en særskilt efterkalkulationsversion, der indeholder de trinvise omkostningsposter.</span><span class="sxs-lookup"><span data-stu-id="8d4a7-109">The two-version approach can be used to identify multiple incremental updates, where each incremental update has a separate costing version that contains the incremental cost records.</span></span>
