[!include [banner](../includes/banner.md)]


<span data-ttu-id="6d71b-104">Du kan oprette dispositionskoder for at angive, hvordan du kan behandle en vare, der er returneret af en debitor.</span><span class="sxs-lookup"><span data-stu-id="6d71b-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="6d71b-105">Du kan f.eks. oprette en dispositionskode, der hedder **Reparation og returnering**, til at angive, at den returnerede vare skal repareres og derefter returneres til debitoren.</span><span class="sxs-lookup"><span data-stu-id="6d71b-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="6d71b-106">Du kan finde flere eksempler på dispositionskoder, der typisk bruges til varer, der returneres af debitorer, under [Angive, hvordan returvarer skal kasseres](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="6d71b-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="6d71b-107">Du kan også angive en årsagskode til at forklare, hvorfor en vare blev returneret.</span><span class="sxs-lookup"><span data-stu-id="6d71b-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="6d71b-108">Du kan finde flere oplysninger om, hvordan du konfigurerer årsagskoder, i [Konfigurer årsagskoder for returnering](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="6d71b-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="6d71b-109">Gå til **Salg og marketing** \> **Opsætning** \> **Salgsordrer** \> **Returneringer** \> **Dispositionskoder**.</span><span class="sxs-lookup"><span data-stu-id="6d71b-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="6d71b-110">Vælg **Ny** for at oprette en ny dispositionskode.</span><span class="sxs-lookup"><span data-stu-id="6d71b-110">Select **New** to create a new disposition code.</span></span>
