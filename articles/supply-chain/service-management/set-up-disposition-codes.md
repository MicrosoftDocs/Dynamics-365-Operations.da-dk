# <a name="set-up-disposition-codes"></a><span data-ttu-id="0f084-103">Konfigurer dispositionskoder</span><span class="sxs-lookup"><span data-stu-id="0f084-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="0f084-104">Du kan oprette dispositionskoder for at angive, hvordan du kan behandle en vare, der er returneret af en debitor.</span><span class="sxs-lookup"><span data-stu-id="0f084-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="0f084-105">Du kan f.eks. oprette en dispositionskode, der hedder **Reparation og returnering**, til at angive, at den returnerede vare skal repareres og derefter returneres til debitoren.</span><span class="sxs-lookup"><span data-stu-id="0f084-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="0f084-106">Du kan finde flere eksempler på dispositionskoder, der typisk bruges til varer, der returneres af debitorer, under [Angive, hvordan returvarer skal kasseres](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="0f084-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="0f084-107">Du kan også angive en årsagskode til at forklare, hvorfor en vare blev returneret.</span><span class="sxs-lookup"><span data-stu-id="0f084-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="0f084-108">Du kan finde flere oplysninger om, hvordan du konfigurerer årsagskoder, i [Konfigurer årsagskoder for returnering](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="0f084-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="0f084-109">Klik på **Salg og marketing** \> **Opsætning** \> **Salgsordrer** \> **Returneringer** \> **Dispositionskoder**.</span><span class="sxs-lookup"><span data-stu-id="0f084-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="0f084-110">Klik på **Ny**, eller tryk på CTRL+N for at oprette en ny dispositionskode.</span><span class="sxs-lookup"><span data-stu-id="0f084-110">Click **New** or press CTRL+N to create a new disposition code.</span></span>

3.  <span data-ttu-id="0f084-111">Skriv et entydigt, beskrivende navn, vælg en handling, og skriv en beskrivelse af dispositionskoden.</span><span class="sxs-lookup"><span data-stu-id="0f084-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="0f084-112">Hvis du vil knytte kundegebyrer til denne dispositionskode, skal du klikke på knappen **Gebyrer** for at åbne formularen **Opret gebyrer**.</span><span class="sxs-lookup"><span data-stu-id="0f084-112">If you want to associate any customer charges with this disposition code, click the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="0f084-113">Hvis du vil definere eksterne koder, der svarer til firmaets egne dispositionskoder, skal du klikke på knappen **Eksterne kode** for at åbne formularen **Eksterne kode**.</span><span class="sxs-lookup"><span data-stu-id="0f084-113">If you want to define any external codes to match with the company's own disposition codes, click the **External codes** button to open the **External codes** form.</span></span>
