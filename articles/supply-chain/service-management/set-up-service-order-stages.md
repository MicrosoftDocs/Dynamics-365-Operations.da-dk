# <a name="set-up-service-order-stages"></a><span data-ttu-id="fc955-103">Konfigurer serviceordrestadier</span><span class="sxs-lookup"><span data-stu-id="fc955-103">Set up service order stages</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="fc955-104">Klik på **Servicestyring** \> **Opsætning** \> **Serviceordrer** \> **Servicestadier**.</span><span class="sxs-lookup"><span data-stu-id="fc955-104">Click **Service management** \> **Setup** \> **Service orders** \> **Service stages**.</span></span>

2.  <span data-ttu-id="fc955-105">Tryk på Ctrl+N for at oprette en ny post.</span><span class="sxs-lookup"><span data-stu-id="fc955-105">Press CTRL+N to create a new record.</span></span>

3.  <span data-ttu-id="fc955-106">Indtast et servicestadie-id og en beskrivelse i felterne **Servicestadie** og **Beskrivelse**.</span><span class="sxs-lookup"><span data-stu-id="fc955-106">In the **Service stage** and **Description** fields, specify a service stage ID and description.</span></span>

4.  <span data-ttu-id="fc955-107">Vælg de rette parametre for stadiet.</span><span class="sxs-lookup"><span data-stu-id="fc955-107">Select the appropriate parameters for the stage.</span></span>

5.  <span data-ttu-id="fc955-108">Vælg det overordnede stadie til det aktuelle stadie, eller lad feltet **Overordnet** være tomt, hvis stadiet er begyndelsesstadiet i stadieopsætningen.</span><span class="sxs-lookup"><span data-stu-id="fc955-108">Select the parent stage for the current stage or leave the **Parent** field blank if the stage is the initial stage in the stage setup.</span></span>
