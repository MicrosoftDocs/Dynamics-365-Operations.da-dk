# <a name="set-up-subscription-groups"></a><span data-ttu-id="32731-103">Konfigurer abonnementsgrupper</span><span class="sxs-lookup"><span data-stu-id="32731-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="32731-104">Gå til **Servicestyring** \> **Opsætning** \> **Serviceabonnementer** \> **Abonnementsgrupper**.</span><span class="sxs-lookup"><span data-stu-id="32731-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="32731-105">Vælg **Ny** for at oprette en ny abonnementsgruppe.</span><span class="sxs-lookup"><span data-stu-id="32731-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="32731-106">I feltet **Abonnementsgruppe** skal du angive et navn til den nye abonnementsgruppe.</span><span class="sxs-lookup"><span data-stu-id="32731-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="32731-107">Vælg en fakturafrekvens til abonnementet i feltet **Fakturering**.</span><span class="sxs-lookup"><span data-stu-id="32731-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="32731-108">Fakturafrekvensen angives med en periodekode, der oprettes i formularen **Periodetyper**.</span><span class="sxs-lookup"><span data-stu-id="32731-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>
