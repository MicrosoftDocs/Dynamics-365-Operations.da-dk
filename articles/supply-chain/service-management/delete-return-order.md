<span data-ttu-id="33011-104">Du kan bruge formularen **Slet returordre** til at slette flere returordrer.</span><span class="sxs-lookup"><span data-stu-id="33011-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="33011-105">Klik på **Salg og markering** \> **Periodisk** \> **Oprydning** \> **Slet returordrer**.</span><span class="sxs-lookup"><span data-stu-id="33011-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>
