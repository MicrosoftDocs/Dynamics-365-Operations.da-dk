# <a name="set-up-service-intervals"></a><span data-ttu-id="19972-103">Oprette serviceintervaller</span><span class="sxs-lookup"><span data-stu-id="19972-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="19972-104">Serviceintervallet angiver den frekvens, hvormed serviceordrelinjer oprettes til serviceaftalelinjer, når du opretter serviceordrer.</span><span class="sxs-lookup"><span data-stu-id="19972-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>
