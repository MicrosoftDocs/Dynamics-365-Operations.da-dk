# <a name="mobile-bar-code-support"></a><span data-ttu-id="a02f9-103">Understøttelse af mobile stregkode</span><span class="sxs-lookup"><span data-stu-id="a02f9-103">Mobile bar code support</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a02f9-104">Da Android er et open source-projekt, kan enhver producent af hardware til stregkodescannere til lagerbrug bygge en enhed, der kan køre Android-operativsystemet.</span><span class="sxs-lookup"><span data-stu-id="a02f9-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="a02f9-105">En enhed er kun Android-kompatibel, hvis den kan køre programmer, der er skrevet til Android-kørselsmiljøet.</span><span class="sxs-lookup"><span data-stu-id="a02f9-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="a02f9-106">En hardwareleverandør kan dog redigere og oprette overlejringer til Android-versionen, der kører på deres hardware.</span><span class="sxs-lookup"><span data-stu-id="a02f9-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="a02f9-107">Microsoft kan ikke tage sig ansvar for at sikre, at en app til mobil stregkodescanning til Android er kompatibel med en producents hardware til stregkodescanning og den Android-version, der kører på den.</span><span class="sxs-lookup"><span data-stu-id="a02f9-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span> 

<span data-ttu-id="a02f9-108">Lager-appen til Microsoft Dynamics 365 for Finance and Operations er blevet testet for scanning af stregkoder på et udvalg af Android-baserede enheder.</span><span class="sxs-lookup"><span data-stu-id="a02f9-108">The Warehousing app for Microsoft Dynamics 365 for Finance and Operations has been tested with a selection of Android powered devices for bar code scanning.</span></span> <span data-ttu-id="a02f9-109">Disse test omfatter kun et udvalg af de enheder, der er tilgængelige på markedet.</span><span class="sxs-lookup"><span data-stu-id="a02f9-109">These tests only cover a sample of the devices that are available on the market.</span></span>

<span data-ttu-id="a02f9-110">Det anbefales, at du som kunde tester appen til mobil scanning til lageret på udvalgt hardware, før du beslutter dig for den hardware, som du vil købe.</span><span class="sxs-lookup"><span data-stu-id="a02f9-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
