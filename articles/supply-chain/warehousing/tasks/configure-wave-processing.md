# <a name="configure-wave-processing"></a><span data-ttu-id="1ce63-103">Konfigurere behandling af bølge</span><span class="sxs-lookup"><span data-stu-id="1ce63-103">Configure wave processing</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="1ce63-104">Denne vejledning beskriver, hvordan du konfigurerer de kriterier, der bestemmer, hviket arbejde der genereres for et lagersted, når en bølge behandles, og om bølger behandles manuelt eller automatisk.</span><span class="sxs-lookup"><span data-stu-id="1ce63-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="1ce63-105">Du angiver kriterierne ved at konfigurere bølgeskabeloner og forespørgsler, der svarer til en bølge med frigivne linjer i salgsordrer, produktionsordrer eller kanbanordrer.</span><span class="sxs-lookup"><span data-stu-id="1ce63-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="1ce63-106">Bølgebehandling bruges på lagersteder, der bruger funktionerne i modulet Lagerstedsstyring, og ikke dem, der bruger funktionerne i modulet Lagerstyring.</span><span class="sxs-lookup"><span data-stu-id="1ce63-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="1ce63-107">Du kan bruge denne procedure i USMF-demodatafirmaet.</span><span class="sxs-lookup"><span data-stu-id="1ce63-107">You can run this procedure in demo data company USMF.</span></span>

3. <span data-ttu-id="1ce63-110">Skriv en værdi i feltet Bølgeskabelonnavn.</span><span class="sxs-lookup"><span data-stu-id="1ce63-110">In the Wave template name field, type a value.</span></span>
    * <span data-ttu-id="1ce63-111">Når du opretter en bølgeskabelon, kan du angive den rækkefølge, hvori skabelonerne knyttes til frigivne linjer på salgsordrer, produktionsordrer eller kanbans.</span><span class="sxs-lookup"><span data-stu-id="1ce63-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="1ce63-112">Når en linje er frigivet til lagerstedet eller produktionen, bruges den første bølgeskabelon, den opfylder kriterierne for.</span><span class="sxs-lookup"><span data-stu-id="1ce63-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="1ce63-113">Vi anbefaler, at du placerer skabeloner med de mest specifikke kriterier øverst på listen.</span><span class="sxs-lookup"><span data-stu-id="1ce63-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="1ce63-114">Jo bredere kriterier, desto mere sandsynligt er det, at en linje opfylder kriterierne, og et kan medføre, at linjer tildeles til den forkerte bølge.</span><span class="sxs-lookup"><span data-stu-id="1ce63-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>

6. <span data-ttu-id="1ce63-118">Indtast eller vælg en værdi i feltet Lagersted.</span><span class="sxs-lookup"><span data-stu-id="1ce63-118">In the Warehouse field, enter or select a value.</span></span>

9. <span data-ttu-id="1ce63-124">Angiv indstillingen Automatiser frigivelse af bølge til Ja.</span><span class="sxs-lookup"><span data-stu-id="1ce63-124">Set the Automate wave release option to Yes.</span></span> 
    * <span data-ttu-id="1ce63-125">Vælg denne indstilling for at frigive bølgen automatisk.</span><span class="sxs-lookup"><span data-stu-id="1ce63-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="1ce63-126">Plukkearbejdet oprettes og gøres tilgængeligt på mobilenheder.</span><span class="sxs-lookup"><span data-stu-id="1ce63-126">The picking work is created and made available on mobile devices.</span></span>

    * <span data-ttu-id="1ce63-128">Linjerne tildeles bølger ud fra forespørgselsfiltret til bølgeskabelonen.</span><span class="sxs-lookup"><span data-stu-id="1ce63-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="1ce63-129">Angiv indstillingen Udfør automatisk behandling af bølge ved grænseværdi til Ja.</span><span class="sxs-lookup"><span data-stu-id="1ce63-129">Set the Process wave automatically at threshold option to Yes.</span></span> 
    * <span data-ttu-id="1ce63-130">Vælg denne indstilling for automatisk at behandle bølgen, når dens værdier når de grænser for vægt, levering, og linjer, der er angivet i feltgruppen Grænser for bølge.</span><span class="sxs-lookup"><span data-stu-id="1ce63-130">Select this option to automatically process the wave when its values reach the thresholds for weight, shipment, and lines specified in the Wave thresholds field group.</span></span> <span data-ttu-id="1ce63-131">Denne indstilling kan kun benyttes, hvis der er valgt Forsendelse i feltet Bølgeskabelontype.</span><span class="sxs-lookup"><span data-stu-id="1ce63-131">This option is available only if Shipping is selected in the Wave template type field.</span></span>

    * <span data-ttu-id="1ce63-136">Bølgeskabelonmetoder gør det muligt at styre den rækkefølge af aktiviteter, som hver bølge gennemgår når den er behandlet.</span><span class="sxs-lookup"><span data-stu-id="1ce63-136">Wave template methods allow you to control the sequence of activities that each wave is going through when it’s processed.</span></span> <span data-ttu-id="1ce63-137">Du kan for eksempel have en metode til bølgegenopfyldning.</span><span class="sxs-lookup"><span data-stu-id="1ce63-137">For example, you might have a method for wave replenishment.</span></span> <span data-ttu-id="1ce63-138">Når du tilføjer en metode, vises den automatisk på den korrekte placering i trinrækkefølgen.</span><span class="sxs-lookup"><span data-stu-id="1ce63-138">When you add a method, it’s automatically listed in the appropriate location in the sequence of steps.</span></span> <span data-ttu-id="1ce63-139">Hvis du har angivet indstillingen Automatiser frigivelse af genopfyldningsarbejde til Ja, skal du tilføje genopfyldningsmetoden her.</span><span class="sxs-lookup"><span data-stu-id="1ce63-139">If you’ve set the Automate replenishment work release option to Yes, you need to add the replenish method here.</span></span>
