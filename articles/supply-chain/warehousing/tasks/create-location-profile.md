# <a name="create-a-location-profile"></a><span data-ttu-id="9956d-103">Oprette en lokalitetsprofil.</span><span class="sxs-lookup"><span data-stu-id="9956d-103">Create a location profile</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="9956d-104">Hver lokalitet på lagerstedet skal have tilknyttet en lokalitetsprofil, der beskriver egenskaberne for lokaliteten, for eksempel om lokaliteten tillader blandede varer.</span><span class="sxs-lookup"><span data-stu-id="9956d-104">Every location in the warehouse needs to have a location profile associated with it that describes the properties of the location, for example, whether the location allows mixed items.</span></span> <span data-ttu-id="9956d-105">I denne procedure skal vi oprette en profil for en lokalitet, der ikke kræver id-kontrol.</span><span class="sxs-lookup"><span data-stu-id="9956d-105">In this procedure we’ll create a profile for a location that doesn’t require license plate control.</span></span> <span data-ttu-id="9956d-106">Vi aktiverer blandede varer og blandede lagerstatusser og tillader cyklusoptælling.</span><span class="sxs-lookup"><span data-stu-id="9956d-106">We’ll enable mixed items, and mixed inventory statuses, and allow cycle counting.</span></span> <span data-ttu-id="9956d-107">Du kan bruge denne procedure i USMF-demodatafirmaet.</span><span class="sxs-lookup"><span data-stu-id="9956d-107">You can use this procedure in the USMF demo data company.</span></span>

1. <span data-ttu-id="9956d-108">Klik på Ny.</span><span class="sxs-lookup"><span data-stu-id="9956d-108">Click New.</span></span>
2. <span data-ttu-id="9956d-109">Skriv en værdi i feltet Id for lokationsprofil.</span><span class="sxs-lookup"><span data-stu-id="9956d-109">In the Location profile ID field, type a value.</span></span>
3. <span data-ttu-id="9956d-110">Skriv en værdi i feltet Navn.</span><span class="sxs-lookup"><span data-stu-id="9956d-110">In the Name field, type a value.</span></span>
4. <span data-ttu-id="9956d-111">Indtast eller vælg en værdi i feltet Lokationsformat.</span><span class="sxs-lookup"><span data-stu-id="9956d-111">In the Location format field, enter or select a value.</span></span>
5. <span data-ttu-id="9956d-112">Indtast eller vælg en værdi i feltet Lokationstype.</span><span class="sxs-lookup"><span data-stu-id="9956d-112">In the Location type field, enter or select a value.</span></span>
6. <span data-ttu-id="9956d-113">Indtast eller vælg en værdi i feltet Id for dokstyringsprofil.</span><span class="sxs-lookup"><span data-stu-id="9956d-113">In the Dock management profile ID field, enter or select a value.</span></span>
7. <span data-ttu-id="9956d-114">Vælg Ja i feltet Tillad blandede varer.</span><span class="sxs-lookup"><span data-stu-id="9956d-114">Select Yes in the Allow mixed items field.</span></span>
8. <span data-ttu-id="9956d-115">Vælg Ja i feltet Tillad blandede lagerstatusser.</span><span class="sxs-lookup"><span data-stu-id="9956d-115">Select Yes in the Allow mixed  inventory statuses field.</span></span>
9. <span data-ttu-id="9956d-116">Vælg Ja i feltet Tillad cyklusoptælling.</span><span class="sxs-lookup"><span data-stu-id="9956d-116">Select Yes in the Allow cycle counting field.</span></span>
10. <span data-ttu-id="9956d-117">Klik på Gem.</span><span class="sxs-lookup"><span data-stu-id="9956d-117">Click Save.</span></span>
11. <span data-ttu-id="9956d-118">Gå til Lagerstedsstyring > Konfiguration > Lagersted > Lokationsprofiler.</span><span class="sxs-lookup"><span data-stu-id="9956d-118">Go to Warehouse management > Setup > Warehouse > Location profiles.</span></span>
