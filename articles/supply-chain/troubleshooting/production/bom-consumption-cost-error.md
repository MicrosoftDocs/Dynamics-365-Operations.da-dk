# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>En produktionsordre kan ikke afsluttes pga. fejl i værdien styklisten consumptionCos

## <a name="symptoms"></a>Symptomer

Når du forsøger at afslutte en produktionsordre, får du vist følgende fejlmeddelelse:

> Beregningen af værdien for BOM consumptionCost skal være negativ ved lagertil afgang.

## <a name="resolution"></a>Løsning

Dette problem blev løst i version 10.0.15.
