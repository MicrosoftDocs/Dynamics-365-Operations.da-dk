# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Brugere kan fortryde plukning på materialelinjer for produkter, der er blevet færdigmeldt

KB-nummer: 4614721

## <a name="symptoms"></a>Symptomer

Systemet gør det muligt for brugerne, at fortryde plukning på råmaterialelinjer for produkter, der er blevet færdigmeldt.

## <a name="resolution"></a>Løsning

Systemet fungerer som designet. Råvareforbruget kan reguleres, indtil produktionsordren når statussen *Afsluttet*.
