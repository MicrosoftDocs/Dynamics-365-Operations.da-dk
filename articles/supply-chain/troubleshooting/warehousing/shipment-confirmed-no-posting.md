# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>Forsendelsen til indlæsning er bekræftet, men der er ikke bogført nogen linjer

## <a name="symptoms"></a>Symptomer

Når du arbejder med udgående lagerstedshandlinger, kan du få vist følgende meddelelse:

> Forsendelsen af lasten 1% er blevet bekræftet.

Der er dog ikke foretaget yderligere bogføring.

## <a name="resolution"></a>Løsning

Det er efter design. Forsendelsesbekræftelse påvirker ikke bogføringen. Den opdaterer blot status for afsendelse og last. Bogføring skal finde sted i en separat proces.
