# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>Ugyldigt id-fejl ved scanning af id i mobilappen

## <a name="symptoms"></a>Symptomer

Du får vist denne fejlmeddelelse i mobilappen Warehouse Management, når du scanner et nummerplade-id.

> Ugyldigt id.

## <a name="resolution"></a>Løsning

Kontrollér, at nummerpladens id findes i id-tabellen, og at varerne og antallet på id'et er tilgængelige (med andre ord blokeres de ikke).
