# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="d0fc9-103">Oprette en stregkode for et produkt</span><span class="sxs-lookup"><span data-stu-id="d0fc9-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="d0fc9-104">Dette emne viser, hvordan du manuelt opretter en stregkode med varenummer M0001 som eksempel.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="d0fc9-105">Det demodatafirma, der bruges til at oprette denne procedure, er USMF.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="d0fc9-106">Vælg **Vedligeholdelse af frigivet produkt** på startsiden.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="d0fc9-107">Gå til **Produkter > Frigivne produkter** i afsnittet **Links**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="d0fc9-108">Find og vælg den ønskede post på listen.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="d0fc9-109">I dette eksempel skal du vælge varenummer **M0001**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="d0fc9-110">Vælg **Styr lager** i handlingsruden.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="d0fc9-111">Vælg **Stregkoder**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="d0fc9-112">Vælg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-112">Select **New**.</span></span>
7. <span data-ttu-id="d0fc9-113">Markér den valgte række, der er oprettet på listen nedenfor.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="d0fc9-114">Indtast eller vælg en værdi i feltet **Stregkodeopsætning**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="d0fc9-115">Indtast eller vælg en værdi i feltet **Stregkode**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="d0fc9-116">Skriv en værdi i feltet **Stregkode**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="d0fc9-117">Angiv et tal i feltet **Antal**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="d0fc9-118">Vælg **Gem**.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-118">Select **Save**.</span></span>
13. <span data-ttu-id="d0fc9-119">Luk siden.</span><span class="sxs-lookup"><span data-stu-id="d0fc9-119">Close the page.</span></span>
