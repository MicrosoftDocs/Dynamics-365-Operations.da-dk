# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="90905-103">Oprette en stregkode for et produkt</span><span class="sxs-lookup"><span data-stu-id="90905-103">Create a bar code for a product</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="90905-104">Dette emne viser, hvordan du manuelt opretter en stregkode med varenummer M0001 som eksempel.</span><span class="sxs-lookup"><span data-stu-id="90905-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="90905-105">Det demodatafirma, der bruges til at oprette denne procedure, er USMF.</span><span class="sxs-lookup"><span data-stu-id="90905-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="90905-106">Vælg **Vedligeholdelse af frigivet produkt** på startsiden.</span><span class="sxs-lookup"><span data-stu-id="90905-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="90905-107">Gå til **Produkter > Frigivne produkter** i afsnittet **Links**.</span><span class="sxs-lookup"><span data-stu-id="90905-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="90905-108">Find og vælg den ønskede post på listen.</span><span class="sxs-lookup"><span data-stu-id="90905-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="90905-109">I dette eksempel skal du vælge varenummer **M0001**.</span><span class="sxs-lookup"><span data-stu-id="90905-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="90905-110">Vælg **Styr lager** i handlingsruden.</span><span class="sxs-lookup"><span data-stu-id="90905-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="90905-111">Vælg **Stregkoder**.</span><span class="sxs-lookup"><span data-stu-id="90905-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="90905-112">Vælg **Ny**.</span><span class="sxs-lookup"><span data-stu-id="90905-112">Select **New**.</span></span>
7. <span data-ttu-id="90905-113">Markér den valgte række, der er oprettet på listen nedenfor.</span><span class="sxs-lookup"><span data-stu-id="90905-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="90905-114">Indtast eller vælg en værdi i feltet **Stregkodeopsætning**.</span><span class="sxs-lookup"><span data-stu-id="90905-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="90905-115">Indtast eller vælg en værdi i feltet **Stregkode**.</span><span class="sxs-lookup"><span data-stu-id="90905-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="90905-116">Skriv en værdi i feltet **Stregkode**.</span><span class="sxs-lookup"><span data-stu-id="90905-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="90905-117">Angiv et tal i feltet **Antal**.</span><span class="sxs-lookup"><span data-stu-id="90905-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="90905-118">Vælg **Gem**.</span><span class="sxs-lookup"><span data-stu-id="90905-118">Select **Save**.</span></span>
13. <span data-ttu-id="90905-119">Luk siden.</span><span class="sxs-lookup"><span data-stu-id="90905-119">Close the page.</span></span>
