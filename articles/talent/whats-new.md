# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr"></a>Nyheder eller ændringer i Dynamics 365 for Talent Core HR 

[!include [banner](includes/banner.md)]

På denne side kan du se, hvordan du finder oplysninger om den seneste version af Microsoft Dynamics Talent Core HR.

Hver uge er der nye frigivelser til Talent. Disse frigivelser er angivet i indholdsfortegnelsen på webstedet docs.microsoft.com ([docs.microsoft.com/dynamics365/unified-operations](../index.md)). Du kan enten scanne indholdsfortegnelsen eller bruge kontrolelementet Filter over indholdsfortegnelsen til at finde de seneste emner, du søger efter. Titlerne på emnerne begynder med "Nyheder eller ændringer i Dynamics 365 for Talent...".

Indholdsfortegnelsen bliver løbende opdateret hver uge, så kom tilbage for at finde alle de nye funktioner og ændringer i de seneste frigivelser til Talent.
