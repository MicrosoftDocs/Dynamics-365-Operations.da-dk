# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Nyheder eller ændringer i Dynamics 365 Talent (17. december 2019)

I denne artikel beskrives funktioner, der enten er nye eller ændrede i Dynamics 365 Talent.

## <a name="changes-in-attract"></a>Ændringer i Attract

Denne version indeholder rettelser af mindre fejl i Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Ændringer i Onboard

Denne version indeholder rettelser af mindre fejl i Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Ændringer i Core HR

Ændringer, der er beskrevet i dette afsnit, gælder for build-nummer 8.1.2674. Tallene i parenteser i nogle overskrifter henviser til supportnumre i Microsoft Dynamics Lifecycle Services (LCS).

### <a name="print-performance-reviews"></a>Udskiv performancegennemgange

Udskrivning af performancegennemgange er nu tilgængelig i alle miljøer. Hvis du vil aktivere denne funktion, skal du bruge arbejdsområdet **Funktionsstyring**. Du kan finde flere oplysninger om Administration af funktioner under [Oversigt over funktionsstyring](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Du finder flere oplysninger om udskrivning af performancegennemgange i [Performancegennemgange](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Pensioneret stilling med direkte rapporter "flyder" i org-diagram som en åben position (385436)

Denne uges udgivelse korrigerer, hvordan pensionerede stillinger med direkte rapporter vises i rapporteringshierarkiet.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>Systemet kan ikke generere kalenderdage (381195) 

Denne uges ændringer korrigerer et problem, hvor kalenderdage ikke kunne genereres.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Aktuel saldorapport i orlov-PBI fordobler medarbejdersaldo (385581)

Denne ændring korrigerer den aktuelle saldorapport i dette scenario.
