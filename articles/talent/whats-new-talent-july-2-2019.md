# <a name="whats-new-or-changed-in-dynamics-365-for-talent-july-02-2019"></a>Nyheder eller ændringer i Dynamics 365 for Talent (2. juli 2019)

[!include [banner](includes/banner.md)]

I dette emne beskrives funktioner, der enten er nye eller ændrede i Microsoft Dynamics 365 for Talent.

## <a name="changes-in-attract"></a>Ændringer i Attract

### <a name="use-organization-service-accounts-to-send-e-mails"></a>Brug organisationstjenestekonti til at sende mails

Du kan nu konfigurere mailindstillinger, så mails til kandidater, leverandører, rekrutteringsmedarbejdere og ansættelseschefer sendes fra din organisations mailtjenestekonto. Du kan finde flere oplysninger under [Forbinde en mailtjenestekonto](./attract-configure-email-settings.md#connect-an-email-service-account).

#### <a name="offer-email-templates"></a>Tilbyde mailskabeloner 

Du kan nu konfigurere organisationens mailskabeloner for alle stadier i forbindelse med udarbejde, godkendelse og accept af tilbud. Brugerne kan enten bruge standardskabelonen eller tilpasse den, før der sendes mail. Du kan få flere oplysninger under [Konfigurere mailskabelonindstillinger](./attract-configure-email-settings.md#configure-email-template-settings).

### <a name="candidate-consent-on-application"></a>Kandidatens samtykke ved ansøgning

Du kan nu aktivere linket for vilkår og betingelser for din organisation. Når denne indstilling er aktiveret, skal ansøgere give deres samtykke til vilkårene og betingelserne, før de sender deres jobansøgninger. 

## <a name="changes-in-onboard"></a>Ændringer i Onboard

Denne version indeholder rettelser af mindre fejl i Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Ændringer i Core HR

Denne version indeholder rettelser af mindre fejl i Dynamics 365 Talent: Core HR.
