<span data-ttu-id="89df4-106">Forkert konfiguration og opsætning af et modul kan påvirke tilgængeligheden af funktioner i Finance and Operations, systemets ydeevne og den glidende afvikling af forretningsprocesser.</span><span class="sxs-lookup"><span data-stu-id="89df4-106">Incorrect configuration and setup of a module can adversely affect the availability of features in Finance and Operations, system performance, and the smooth operation of business processes.</span></span> <span data-ttu-id="89df4-107">Kvaliteten af virksomhedens data (f.eks. korrekte, komplette og rene data) påvirker også systemets ydeevne, og en organisations muligheder for at tage beslutninger, produktivitet og så videre.</span><span class="sxs-lookup"><span data-stu-id="89df4-107">The quality of business data (for example, the correctness, completeness, and cleanliness of the data) also affects system performance, and an organization's decision-making capabilities, productivity, and so on.</span></span>

<span data-ttu-id="89df4-108">Arbejdsområdet **Optimeringsrådgiver** er et værktøj, der giver superbrugere, virksomhedsanalytikere, funktionelle konsulenter og it-supportfunktioner mulighed for at identificere problemer i modulkonfiguration og forretningsdata.</span><span class="sxs-lookup"><span data-stu-id="89df4-108">The **Optimization advisor** workspace is a tool that lets power users, business analysts, functional consultants, and IT support functions identify issues in module configuration and business data.</span></span> <span data-ttu-id="89df4-109">Optimeringsrådgiver foreslår bedste fremgangsmåder til konfiguration af modulet og identificerer forretningsdata, der er forældede eller fejlagtige.</span><span class="sxs-lookup"><span data-stu-id="89df4-109">Optimization advisor suggests best practices for module configuration and identifies business data that is obsolete or incorrect.</span></span>

<span data-ttu-id="89df4-110">Optimeringsrådgiver kører regelmæssigt et sæt regler for bedste fremgangsmåde.</span><span class="sxs-lookup"><span data-stu-id="89df4-110">Optimization advisor periodically runs a set of best practice rules.</span></span> <span data-ttu-id="89df4-111">Der frigives et standardsæt af regler sammen med Microsoft Dynamics 365 for Finance and Operations version 8.0 (april 2018).</span><span class="sxs-lookup"><span data-stu-id="89df4-111">A default set of rules is released together with Microsoft Dynamics 365 for Finance and Operations version 8.0 (April 2018).</span></span> <span data-ttu-id="89df4-112">Brugerne kan også oprette regler, der er specifikke for deres tilpasninger, løsninger fra uafhængige softwareleverandører (ISV'er) og forretningsdata.</span><span class="sxs-lookup"><span data-stu-id="89df4-112">However, users can also create rules that are specific to their customizations, solutions from independent software vendors (ISVs), and business data.</span></span> <span data-ttu-id="89df4-113">Du kan finde flere oplysninger om, hvordan du opretter regler, under [Oprette nye regler](./create-rules-optimization-advisor.md).</span><span class="sxs-lookup"><span data-stu-id="89df4-113">For more information about how to create rules, see [Create new rules](./create-rules-optimization-advisor.md).</span></span>

<span data-ttu-id="89df4-116">Mulighederne kan være firmaspecifikke eller gælde på tværs af firmaer, afhængigt af typen af konfiguration og de data, der skal valideres.</span><span class="sxs-lookup"><span data-stu-id="89df4-116">Opportunities can be company-specific or cross-company, depending on the type of setup and data that is being validated.</span></span> <span data-ttu-id="89df4-117">Muligheder på tværs af firmaer kan ses fra alle firmaer.</span><span class="sxs-lookup"><span data-stu-id="89df4-117">Cross-company opportunities can be viewed from all companies.</span></span> <span data-ttu-id="89df4-118">Hvis du vil have vist muligheder for et specifikt firma, skal du først vælge firmaet.</span><span class="sxs-lookup"><span data-stu-id="89df4-118">To view the opportunities for a specific company, you must first select the company.</span></span>

<span data-ttu-id="89df4-121">Når du udfører en handling i forbindelse med visse optimeringsmuligheder, beregner systemet virkningen af muligheden med hensyn til reduktionen i kørselstiden for forretningsprocesser.</span><span class="sxs-lookup"><span data-stu-id="89df4-121">When you take action on some optimization opportunities, the system calculates the impact of the opportunity in terms of the reduction in the runtime of business processes.</span></span> <span data-ttu-id="89df4-122">Desværre er denne funktion ikke tilgængelig for alle optimeringsmuligheder.</span><span class="sxs-lookup"><span data-stu-id="89df4-122">Unfortunately, this feature isn't available for all optimization opportunities.</span></span>

<span data-ttu-id="89df4-125">Hvis du vil have vist den komplette liste over Optimeringsrådgiver-regler og se, hvor ofte reglerne evalueres, skal du gå til **Systemadministration** &gt; **Periodiske opgaver** &gt; **Vedligehold valideringsregel i diagnosticering**.</span><span class="sxs-lookup"><span data-stu-id="89df4-125">To view the complete list of Optimization advisor rules and to see how often the rules are evaluated, go to **System administration** &gt; **Periodic tasks** &gt; **Maintain diagnostics validation rule**.</span></span> <span data-ttu-id="89df4-126">Kun de regler, der har status som **Aktiv** evalueres.</span><span class="sxs-lookup"><span data-stu-id="89df4-126">Only rules that have a status of **Active** are evaluated.</span></span> <span data-ttu-id="89df4-127">Evalueringsfrekvensen kan angives til **Daglig**, **Ugentlig**, **Månedlig** eller **Ikke-planlagt**.</span><span class="sxs-lookup"><span data-stu-id="89df4-127">The evaluation frequency can be set to **Daily**, **Weekly**, **Monthly**, or **Unscheduled**.</span></span>

<span data-ttu-id="89df4-128">For at udløse evalueringen af ikke-planlagte regler eller for at evaluere periodiske regler igen uden for deres foruddefinerede interval, skal du gå til **Systemadministration** &gt; **Periodiske opgaver** &gt; **Planlæg valideringsregel i diagnosticering**.</span><span class="sxs-lookup"><span data-stu-id="89df4-128">To trigger the evaluation of unscheduled rules, or to reevaluate periodic rules outside their predefined schedule, go to **System administration** &gt; **Periodic tasks** &gt; **Schedule diagnostics validation rule**.</span></span> <span data-ttu-id="89df4-129">Vælg en frekvens for evaluering i dialogboksen **Validering af diagnosticeringsregel**.</span><span class="sxs-lookup"><span data-stu-id="89df4-129">Then, in the **Diagnostic rule validation** dialog box, select an evaluation frequency.</span></span> <span data-ttu-id="89df4-130">Alle regler, der har den angivne frekvens, evalueres igen.</span><span class="sxs-lookup"><span data-stu-id="89df4-130">All rules that have the specified frequency will be reevaluated.</span></span>

<span data-ttu-id="89df4-147">Hvis du ikke kører nogle af forretningsprocesserne i overensstemmelse med de bedste fremgangsmåder (hvis du f.eks. kører forudgående lagerlukning, før lageret er lukket, eller hvis du bruger planlagt batch til batchoverførsel af reskontrokladde), informerer optimeringsmulighederne dig om den bedste fremgangsmåde og beder dig om at følge den.</span><span class="sxs-lookup"><span data-stu-id="89df4-147">If you aren't running some business processes according to best practices (for example, if you run inventory pre-closing before the inventory is closed, or if you use the scheduled batch for subledger journal batch transfer), optimization opportunities inform you about the best practice and ask that you follow it.</span></span>
